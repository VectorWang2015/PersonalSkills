# 股票估值综合分析

## 定位

聚合 skill，用于二级市场股票标的估值。它不替代行业基本面分析，而是在已有财务数据、基本面判断或用户假设之上，选择估值方法、构造估值区间、审计关键假设，并避免把单一倍数或单点 DCF 误当成结论。

核心原则：**先分型，再选方法；先给区间，再讲假设；先审计一致性，再谈便宜或昂贵。**

---

## 输入要求

分析前尽量确认：

1. **股票名称/代码 + 报告期**，例如“301101 明月镜片 2025 年报 + 2026Q1”。
2. **数据来源**：优先为结构化财报解析目录、公司研究报告、估值输入表、用户粘贴的关键数据。
3. **市场数据可用性**：当前股价、市值、股本、净现金/净债务、同行估值是否可用。
4. **用户想要的输出**：合理市值区间、每股目标价区间、估值方法选择、估值模型审计，或反向 DCF。

若缺少实时股价、总股本或当前市值，不得强行计算每股目标价；可输出“合理市值区间/估值框架”，并把缺口列明。

## 解析目录输入兼容

当用户提供 `financial-pdf-parser` 解析目录或类似结构化目录时，先读取：

1. `analysis_context.md`：确认报告来源、报告期、关键表路径和解析警告。
2. `validation/validation_report.md` 或 `.json`：有失败校验的指标必须标注为 provisional。
3. `tables_merged/*.json` 或 `.csv`：优先用于财务数字、利润、现金流、资产负债、股本等。
4. `chunks.jsonl` / `document.md`：只用于业务描述、风险因素、管理层讨论和行业文字。

不得只凭长文本中的数字做估值结论；核心估值输入必须说明来源和数据缺口。

## 执行流水线

```text
第一层：估值对象分型
├─ 成熟稳定股 / 高ROIC成长股 / 亏损成长股 / 周期或商品股
├─ 金融股 / 出口制造 / 医疗器械修复 / 资产重公司 / 困境股
└─ 明确用户问的是内在价值、相对便宜、目标价，还是模型审计

第二层：方法选择
├─ 成熟稳定股：PE 或 EV/EBITDA + 简化 DCF / FCF yield 交叉验证
├─ 高ROIC成长股：DCF 或 PE/PEG + ROIC/再投资一致性检查
├─ 亏损成长股：Top-down DCF + PS/EV/Sales + 存活概率/融资稀释
├─ 周期/商品股：正常化盈利 + PB/EV/产能指标，避免低PE陷阱
├─ 金融股：PB/PE/DDM/excess return，避免普通 EV/EBITDA
├─ 困境股：清算价值/困境DCF/股权期权属性
└─ 数据不足：反向 DCF 或估值输入表，不给精确目标价

第三层：估值输入构造
├─ 基准利润或现金流年份
├─ 保守/基准/乐观三情景
├─ 三情景倍数或折现假设
├─ 净现金/净债务/少数股东权益/股权激励稀释
└─ 关键假设、风险折扣、数据缺口

第四层：一致性审计
├─ FCFE ↔ cost of equity；FCFF ↔ WACC
├─ 增长 ↔ 再投资率 ↔ ROIC/ROE
├─ 终值 ↔ 成熟期增长/风险/资本结构
├─ 倍数 ↔ 增长/风险/利润率/ROE或ROIC
└─ 是否重复计算风险、协同、期权、稀释或折价

第五层：输出估值结论
└─ 合理区间 + 方法理由 + 敏感性 + 不能回答的问题
```

## 股票分型与估值方法

| 股票类型 | 主方法 | 交叉验证 | 关键假设 | 主要陷阱 |
|---|---|---|---|---|
| 成熟稳定股 | PE、FCF yield、简化 DCF | 股息率、EV/EBITDA | 利润稳定性、现金流含金量、分红能力 | 低增长公司给高成长倍数 |
| 高 ROIC 成长股 | DCF、PE/PEG | ROIC、再投资率、收入增长质量 | 增长持续期、利润率、再投资效率 | 只看增速不看资本回报 |
| 亏损成长股 | Top-down DCF、EV/Sales | 单位经济、目标利润率、融资稀释 | TAM、份额、目标 margin、存活概率 | 用 PS 忽略利润率和现金消耗 |
| 周期/商品股 | 正常化 PE、PB、EV/EBITDA | 全周期利润、产能、商品价格情景 | 当前处于周期位置 | 景气顶部低 PE 误判便宜 |
| 金融股 | PB/ROE、PE、DDM、excess return | 资本充足率、拨备、内含价值 | 监管资本、ROE vs cost of equity | 用 EV/EBITDA 或普通 FCFF |
| 出口制造 | 正常化 PE、EV/EBITDA、FCF yield | 汇率、关税、客户集中、营运资本 | 订单持续性和现金回款 | 单季度修复年化过度乐观 |
| 医疗器械修复 | 恢复年利润 PE | 毛利率、集采影响、海外收入、商誉 | 集采后价格和量的恢复路径 | 把政策冲击当一次性噪声 |
| 困境股 | 清算价值、困境 DCF、equity-as-option | 债务到期、资产价值、重组概率 | 有限责任和债务结构 | 把期权价值当安全边际 |

## 核心公式与使用边界

### 1. PE 市值区间

```latex
\text{Equity Value} = \text{Normalized Net Income} \times \text{Justified PE}
```

- 适用：盈利为正、利润质量可审计、可选择正常化利润。
- 边界：周期股必须用全周期正常化利润，不得用景气高点利润。

### 2. EV/EBITDA 桥接股权价值

```latex
\text{Equity Value} = EBITDA \times (EV/EBITDA) + \text{Cash} - \text{Debt} - \text{Minority Interest}
```

- 适用：折旧摊销扭曲利润、资本结构差异较大、经营现金流相对稳定的非金融企业。
- 边界：金融企业不适用；资本开支很重时必须用 FCF 或 DCF 交叉验证。

### 3. FCFF DCF

```latex
V_{firm}=\sum_{t=1}^{n}\frac{FCFF_t}{(1+WACC)^t}+\frac{TV_n}{(1+WACC)^n}
```

- 适用：可预测公司自由现金流的非金融企业。
- 边界：FCFF 必须使用 WACC；终值假设必须与成熟期 ROIC、再投资率和增长率一致。

### 4. 终值纪律

```latex
TV_n=\frac{FCFF_{n+1}}{WACC-g}
```

```latex
\text{Stable Reinvestment Rate}=\frac{g}{ROIC}
```

- 适用：稳定增长阶段。
- 边界：`g` 不能超过长期经济增长约束；若 `ROIC` 低于 `WACC`，增长会毁灭价值。

### 5. PEG 快速检查

```latex
PEG=\frac{PE}{\text{Expected EPS Growth Rate}}
```

- 适用：盈利为正、增长率可持续、风险和利润率相近的成长股粗筛。
- 边界：不适用于亏损、低增长、周期高点、利润反弹型公司。

## 执行规则

1. **不得直接给买入/卖出建议**，除非用户明确要求投资建议；默认输出研究结论和估值区间。
2. **没有股本/当前股价/当前市值时，不给每股目标价**。
3. **有 parser 校验失败时，相关指标标为 provisional**。
4. **三情景估值优先于单点估值**。
5. **相对估值必须说明可比样本和基本面差异**；没有同行数据时只能做“假设倍数区间”。
6. **DCF 必须说明终值占比和关键敏感性**。
7. **周期股必须做正常化**，不得用当前低 PE 直接判定低估。
8. **金融股必须进入金融估值分支**，不得用普通企业 EV/EBITDA。
9. **估值结论必须区分事实、假设、判断**。

## 边界

### 不适用

- 只问“ROE 多少”“收入增长多少”等单一指标。
- 要求短线预测、技术分析、资金流、K 线形态。
- 非股票资产估值，如房地产项目、私营公司交易、矿权单独估值。
- 没有任何财务数据却要求精确合理价。

### 最常见错误

- 把 PE 低当作低估，而不看周期位置和利润正常化。
- 把 PS 低当作低估，而不看目标利润率和现金消耗。
- 用同行平均倍数，却不解释 ROIC、增长和风险差异。
- 用 DCF 算出单点目标价，却不做终值和折现率敏感性。
- 对同一风险同时下调现金流、提高折现率、降低倍数，重复惩罚。
