# peg-valuation: PEG估值法

## 元数据
- 来源: 彼得·林奇的成功投资 第10章、第13章
- 类型: 估值工具
- 依赖: six-stock-types
- 产出: 股价是否合理/高估/低估的定量判断

## R (Reading): 原文引用

> "任何一家公司股票如果定价合理的话, 市盈率就会与收益增长率相等。"

> "如果一家公司的股票市盈率只有收益增长率的一半, 那么这只股票赚钱的可能性就相当大; 如果股票市盈率是收益增长率的两倍, 那么这只股票亏钱的可能就非常大。"

> "股票价格的走势线与收益线的波动是并驾齐驱的, 如果股票价格线的波动偏离了收益线, 它迟早还会回到与收益线相一致的趋势上。"

> "考虑股息时: (长期收益增长率 + 股息收益率) ÷ 市盈率。小于1较差, 1.5不错, 2倍或更高则是极佳标的。"

## I (Interpretation): 方法论骨架

**PEG = 市盈率(P/E) ÷ 收益增长率(G)**

核心规则:
- PEG = 1: 定价合理
- PEG < 0.5: 显著低估, 买入机会大
- PEG > 2: 显著高估, 大概率亏损

**三步估值法**:

### 1. 基础PEG计算
```
PEG = 当前市盈率 / 预期收益增长率(%)
```
- 市盈率取近12个月或预期市盈率
- 收益增长率取未来2-3年预期或近3-5年历史平均

### 2. 含股息调整公式
```
(PEG_adj) = (增长率 + 股息收益率) / 市盈率
```
- 结果 < 1: 较差
- 结果 = 1.5: 不错
- 结果 ≥ 2: 极佳

### 3. 净现金调整
每股净现金 = (现金 - 长期负债) / 流通股数

调整后买入价 = 股价 - 每股净现金

然后用调整后价格重新计算市盈率, 再进行PEG比较。

**林奇的安全边际**: 偏爱20-25%增长率的公司, 对50%以上高增长持怀疑态度(不可持续), 尤其当这种高增长来自热门行业时。

## A1 (Past Application): 书中案例

1. **福特汽车(1987)**: 股价38美元, 每股净现金16.30美元, 金融服务业务值16.60美元, 汽车业务实际买入成本仅5.10美元(预期EPS 7美元, PE=0.73倍!)
2. **凯玛特**: PE=10(35美元/EPS 3.50), 判断是否合理取决于公司增长率
3. **雅芳**: PE达50倍但增长率仅15%, PEG显著>3, 严重高估, 后大跌
4. **电子数据系统(EDS)**: PE=500倍, 即使15年后盈利增长20倍, 股价仍回到原点, 说明极端高市盈率的毁灭性

## A2 (Future Trigger): 调用条件

用户在以下场景应调用:
- "这家公司现在贵不贵?"
- "市盈率15倍是高还是低?"
- "怎么判断股价是否合理?"
- 用户拿到一家公司的市盈率和增长率数据进行评估时

不适用于: 无盈利的公司(PEG无意义); 周期型公司高峰期低PE的误导; 金融企业(资产负债表估值逻辑不同); 隐蔽资产型公司(资产而非收益驱动估值)。

## E (Execution): 执行步骤

1. **获取数据**:
   - 当前市盈率(来源: 季报/年报/金融数据终端)
   - 未来2-3年收益增长率预期(分析师一致预期)或近3-5年历史复合增长率
   - 股息收益率(如适用)
   - 每股净现金(现金-长期负债)/总股本

2. **计算PEG**:
   - 基础PEG = PE / 增长率
   - 含股息PEG = (增长率 + 股息收益率) / PE
   - 如有大量净现金, 用调整后PE重新计算

3. **分类型解读**:
   - 缓慢增长型: PEG通常较高, 主要看股息
   - 稳定增长型: PEG=1-1.5合理
   - 快速增长型: PEG=0.5-1值得买入
   - 周期型: 注意高峰期PE低=卖出信号, 低谷期PE高=买入信号
   - 困境反转型: PEG不适用, 看负债/现金
   - 隐蔽资产型: PEG不适用, 看资产价值

4. **做出判断**: 综合PEG、净现金、类型特征, 给出"低估/合理/高估"结论

## B (Boundary): 边界与失效

1. **增长率预测的本质局限**: 未来增长率是假设, 与事实总有偏差。当实际增长率远低于预期, PEG参考系崩塌
2. **PEG不能单独使用**: 需结合six-stock-types分类; 快速增长型公司PEG低可能是增长率即将崩溃
3. **周期型公司的PEG陷阱**: 盈利高峰期PE最低→PEG看起来极有吸引力→实际是卖出信号
4. **负增长/无盈利**: PEG公式失效, 应转用资产估值或困境反转分析
5. **A股修正**: A股长期PEG中枢可能高于美股, 需行业横向比较, 不要机械套用
6. **作者盲点**: 林奇在自己的投资中并不严格按PEG操作, PEG更多是参考框架而非铁律
