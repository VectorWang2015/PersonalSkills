# dangerous-stocks: 6种应避开的危险股票

## 元数据
- 来源: 彼得·林奇的成功投资 第9章
- 类型: 风险识别/负面清单
- 依赖: 无
- 产出: 是否应该避开某只股票的判定

## R (Reading): 原文引用

> "如果说有一种股票我避而不买的话, 那它一定是最热门行业中最热门的股票……热门股票上涨得很快, 总是会上涨到远远超过任何估值方法能够估计出来的价值, 但是由于支撑股价快速上涨的只有投资者一厢情愿的期望, 而公司基本面的实质性内容却像高空的空气一样稀薄, 所以热门股跌下去和涨上来的速度一样快。"

> "那些盈利不错的公司通常不是把赚来的钱用于回购股票或者提高分红, 而是更喜欢把钱浪费在愚蠢的收购兼并上。这种一心一意想要"多元化"的公司寻找的是具备以下特点的收购对象: ①收购价格过于高估; ②经营业务完全超出人们的理解范围。"

> "一家好公司的名字单调乏味, 最初会让投资者闻而远之, 而一家资质平平的公司如果名字起得花里胡哨却可以吸引投资者买入, 并给他们一种错误的安全感。"

## I (Interpretation): 方法论骨架

6条回避规则构成负面筛选清单, 在买入任何股票前对照检查。如果触发任何一条, 不等于绝对不能买, 但必须给予更高标准的研究验证。

### 6种应避开的危险股票

#### 1. 热门行业的热门股
- 特征: 每个投资者都在谈论, PE高到没有估值方法能支持
- 危险: 涨得快跌得更快, 一旦下跌不回停顿
- 案例: 家庭购物网络(16个月: 3→47→3.5美元), 地毯业(200家竞争者涌入)
- 机制: 热门行业吸引大量竞争者→价格战→全行业利润消失

#### 2. 被吹捧为"下一个XX"的公司
- 特征: "下一个IBM""下一个麦当劳""下一个迪士尼"
- 危险: 被捧为"下一个"的公司几乎从未成为真正的明星; 且表明原"楷模"也气数将尽
- 案例: "下一个美国玩具反斗城"儿童世界(步履维艰), "下一个Circuit City"五家公司(市值下跌59-96%)

#### 3. "多元恶化"的公司
- 特征: 不断高溢价收购与主业无关的业务
- 危险: 高价买入→经营不善→低价剥离, 财富从大股东转移到小公司股东
- 案例: 美孚收购Marcor零售(赔钱), 吉列收购数字手表(灾难)
- 对比成功: Melville公司(鞋→CVS药品→Marshall's服装, 有协同效应的收购)
- 对比最成功: 埃克森石油(拒绝多元, 只回购股票) — 股价翻倍

#### 4. "小声耳语"的股票
- 特征: 经纪人压低声音推荐的"黑马", 讲述极具诱惑的故事但无业绩支撑
- 危险: 情绪感染力超过理性分析, 几乎从来不成功
- 案例: KMS工业(40→2.5美元, 每换一个热门概念), 林奇本人Bioresponse亏损
- 警句: "听到咝咝声却没有牛排"

#### 5. 过度依赖大客户的供应商
- 特征: 25-50%的商品卖给同一个客户
- 危险: 客户随时可能取消订单或自产, 且大客户有定价谈判强优势
- 案例: SCI Systems依赖IBM, Tandon依赖少数大客户

#### 6. 名字花里胡哨的公司
- 特征: 名字中有"高级""主要""微型""x", 或用首字母缩写
- 危险: 名字引人注目 + 业务无实质 = 虚假安全感
- 反面案例: 瓶盖瓶塞封口公司如果改名为CroCorSea, 就失去了被忽视的优势

## A1 (Past Application): 书中案例

1. **家庭购物网络(Home Shopping Network)**: 典型热门股, 3→47→3.5美元, 林奇警示"人们最多能戴多少条锆石项链?"
2. **地毯行业**: 分析师说"永远增长"时, 200家竞争者已蜂拥而入
3. **Genesco vs Melville**: 同为鞋业公司, Genesco 17年150次收购→破产, Melville每次收购先成功再追加→股价涨30倍
4. **施乐 vs 菲利普·莫里斯**: 施乐(高增长行业+热门股)→20年跌84%; 菲利普·莫里斯(负增长烟草行业)→涨6.4倍

## A2 (Future Trigger): 调用条件

用户在以下场景应调用:
- "这只股票看起来很热, 我该买吗?"
- 听到"下一个XX"标签时
- "这家公司一直在收购, 是好是坏?"
- 经纪人压低声音推荐某"黑马"时
- 发现客户过度集中的公司时

不适用于: 公司管理层主动剥离亏损业务的瘦身期(可能正是困境反转机会); 已经持有且基本面良好的公司日常监测。

## E (Execution): 执行步骤

1. **逐条对照**: 拿候选公司与6条比对, 记录触发项
2. **触发任意一条, 提高研究标准**: 
   - 如果热门 → PEG必须显著低估才考虑
   - 如果"下一个XX" → 对标原型是否还在增长? 模仿者壁垒何在?
   - 如果多元收购 → 检查收购是否在同一价值链, 协同效应是否为真
   - 如果小声耳语 → 等公司用业绩证明自己, 不急IPO
   - 如果客户集中 → 评估客户自产概率和替代成本
   - 如果名字花哨 → 再多看一步实质
3. **触发两条以上**: 强烈建议放弃, 除非有压倒性的利好证据
4. **与perfect-stock-traits交叉检验**: 6条危险信号的每一面, 往往是13条准则中对应的优点的反面

## B (Boundary): 边界与失效

1. **不是绝对的禁买令**: 热门行业中也有赢家(如微软), 但在热潮中买入代价太大
2. **A股差异**: 
   - 中国市场的"概念炒作"比美国更猛, 热门股陷阱更危险
   - "多元恶化"在中国常与国企改革/资产注入混淆, 需区分
   - 名字花哨那条在中国文化背景中需调整(中文名没有英语前缀)
3. **错失机会风险**: 严格回避可能导致错过早期投资机会(如亚马逊早期也是热门概念)
4. **作者盲点**: 林奇自己是避开高科技股的代表, 策略性地放弃了一个时代的重大机会
