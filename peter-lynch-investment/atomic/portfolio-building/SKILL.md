# portfolio-building: 投资组合构建原则

## 元数据
- 来源: 彼得·林奇的成功投资 第16章
- 类型: 决策框架
- 依赖: six-stock-types, peg-valuation
- 产出: 最佳持股数量和组合结构建议

## R (Reading): 原文引用

> "以我个人的观点, 投资者应该尽可能多地持有符合以下条件的股票: ①你个人工作或生活的经验使你对这家公司有着特别深入的了解; ②通过一系列的标准进行检查你发现这家公司具有令人兴奋的远大发展前景。"

> "你通过认真研究结果发现符合这些条件的也许只有一只股票, 也许会有十来只股票, 股票数量的多少无关紧要, 关键是质量高低。"

> "只是为了投资组合多元化而分散投资于一些你并不了解的公司股票是毫无益处的, 对小投资者来说愚蠢的多元化投资所导致的后果非常恐怖、非常可怕。"

> "如果你不能说服自己坚持'当我的股票下跌25%时我就追加买入'的正确信念, 永远戒除'当我的股票下跌25%我就卖出'的毁灭性错误信念, 那么你永远不可能从股票投资上获得什么像样的回报。"

> "把资金永远留在股票市场里, 根据基本面的变化把资金在不同股票之间进行转换。"

## I (Interpretation): 方法论骨架

### 一、持股数量

**核心原则: 3-10只股票, 质量 > 数量**

- 太少(1-2只): 黑天鹅风险太大, 即使是最好公司的股票也可能因意外事件暴跌
- 太多(>20只): 超出业余投资者跟踪能力, 沦为"愚蠢的多元化"
- 最优: 3-10只, 取决于你的研究深度和生活经验覆盖面
- 林奇本人持有1400只, 但这只适用于职业基金经理, 对业余投资者不适用

**为什么多持有几只有利**:
1. 增加遇到10倍股的概率 — 你永远想不到哪只会是真正的10倍股
2. 增大资金在不同股票间调仓的弹性

### 二、组合类型结构

林奇的组合构成:
- 30-40%: 快速增长型(追逐10倍股)
- 10-20%: 稳定增长型(防御性仓位)
- 10-20%: 周期型(时机性仓位)
- 其余: 困境反转型(高风险高回报)
- 少量: 隐蔽资产型

对业余投资者的简化版:
- 2-3只快速增长型(进攻)
- 1-2只稳定增长型(防守)
- 0-1只周期型(时机性)
- 0-2只困境反转型/隐蔽资产型(特殊机会)

**风险排序**(低到高):
缓慢增长型 < 稳定增长型 ≈ 隐蔽资产型(若判断正确) < 周期型(取决于时机准确度) < 快速增长型 ≈ 困境反转型

### 三、调仓原则

1. **永远满仓, 换股不换现金**: 不在股票之间预留现金, 通过换股实现调仓
2. **卖出标准**: 股价涨幅远超基本面改善 → 卖出换入未涨的同类股票
3. **加仓原则**: 股价下跌但基本面变好 → 加仓; 股价下跌且基本面恶化 → 卖出
4. **绝不设止损单**: "当股价下跌到止损价位……随后股票就会强劲反弹"
5. **不要拔掉鲜花浇灌野草**: 别卖涨得好的买跌得多的
6. **更替标准**: 用基本面更好的替换基本面恶化的, 无关股价

### 四、年龄与风险适配

- 年轻人: 可承担更多快速增长型和困境反转风险
- 年长者: 应偏重稳定增长型获取稳定收入

## A1 (Past Application): 书中案例

1. **Stop & Shop**: 林奇最初以为只是30-40%回报的股息股, 结果成了10倍股 — "最终上涨最多的股票往往出乎我意料"
2. **6次30% vs 1次4倍**: 通过在不同稳定增长型股票间调仓, 累积复利效果接近一只4倍股
3. **林奇的组合结构**: 30-40%增长股 + 10-20%稳定增长 + 10-20%周期 + 其余困境反转
4. **克莱斯勒15倍回报**: 困境反转型高风险+高回报的经典案例

## A2 (Future Trigger): 调用条件

用户在以下场景应调用:
- "我该持有几只股票?"
- "怎么构建一个合理的投资组合?"
- "我的持仓是不是太分散/太集中了?"
- "某只股票亏了要不要补仓?"
- 用户刚确定了几只候选股票, 需要分配资金

不适用于: 已有专业化投资策略(如纯价值投资、指数投资); 资金量极小(<1万元); 短线/日内交易; 基金选择建议。

## E (Execution): 执行步骤

1. **盘点现有持股**: 列出所有持仓, 用six-stock-types分类
2. **评估研究深度**: 对每只股票打分 — 是否真正了解? 买入理由是否仍然成立?
3. **检查类型分布**: 是否过度集中在某一类型? 是否有防御性仓位?
4. **执行调仓**:
   - 基本面恶化 → 卖出换入更优标的
   - 股价大幅上涨但基本面未变 → 考虑减仓
   - 股价下跌但基本面改善 → 加仓
   - 发现更好标的 → 替换现有持仓中的最弱标的
5. **定期复盘**: 每季度至少一次, 重新评估每只股票的持有理由

## B (Boundary): 边界与失效

1. **资金量级**: 10万以下资金, 3-5只足够; 百万以上, 可适当增加到10-15只
2. **A股差异**: 
   - A股波动性更大, 林奇"永远满仓"策略的心理压力极高
   - A股打新/配股等行为需要预留部分现金
3. **不是万能组合公式**: 上述百分比是林奇个人风格, 不应机械模仿。关键是"只持有你真正了解的"
4. **作者盲点**: 
   - 林奇是职业基金经理, 他的"业余投资者组合"建议是理论推演, 并非他本人的实操
   - 1400只 vs 3-10只的建议之间存在张力
