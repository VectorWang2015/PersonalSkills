# 风险加权资产减值准备率——更全面的风险抵补衡量指标

## R — 原文 (Reading)

> 笔者一直在思考，到底用什么指标才能完美衡量银行的风险应对能力？最终，我找到了风险加权资产减值准备率。风险加权资产减值准备率=资产减值准备÷风险加权资产。……这一指标和原来的指标相比有显著的优势：首先，风险加权资产覆盖了资产负债表里所有的资产，没有遗漏。其次，用风险加权资产做分母充分考虑了各类资产风险的差异。这个指标越大，说明银行的风险抵补能力就越强；数值越小，说明银行的减值准备不够充分，未来需要计提更多的减值来抵御风险。
>
> — 价投谷子地，《看透银行》6.2.6节

> 银行的资产减值准备中90%是针对贷款资产的。但是实际上银行的贷款占资产的比例多数都不到70%，剩下30%～50%的资产是同业资产、应收投资等。诚然，这些资产中确实包含很多基本无风险的资产，如国债、货币基金等，但是也不乏兴业银行这样在应收投资中放了八九千亿元类贷款的银行。那么，如果单单用衡量贷款减值准备这两个指标来评估风险抵补，显然有失公允。
>
> — 价投谷子地，《看透银行》6.2.6节

> 不良贷款覆盖率中，贷款减值准备是一个客观值，而不良贷款余额是一个主观值，使不良贷款覆盖率这一指标受主观影响较大。与之相反的是，拨贷比这个指标又完全忽略了不同性质贷款产生不良的概率有显著的差异……如果用拨贷比去衡量银行应对风险的能力，对于从事低风险贷款的银行显然也是不公平的。
>
> — 价投谷子地，《看透银行》6.2.6节

## I — 方法论骨架 (Interpretation)

风险加权资产减值准备率是作者独创的风险抵补评估指标，旨在弥补传统两项指标的固有缺陷。其核心逻辑链为：

**第一环：传统两项指标的"三重缺陷"。**

拨备覆盖率 = 拨备÷不良贷款余额。缺陷：(1) 分母只含贷款，遗漏了银行30%-50%的非贷款资产（同业资产、应收投资等），对投资类资产占比高的银行（如兴业在应收投资中放七八千亿元类贷款）评估严重失真；(2) 不良贷款余额是银行认定的"主观值"——不良偏离度从80%到180%不等，这意味着同样金额的减值准备，被认定不良少的银行覆盖率就高，本质上是在"分母造假"。拨贷比 = 拨备÷贷款总额。缺陷：完全无视不同性质贷款的违约概率差异（按揭贷款的违约率远低于对公贷款），对从事低风险零售贷款的银行（如招行）不公平。

**第二环：风险加权资产减值准备率的"三重优势"。**

计算公式 = 资产减值准备 ÷ 风险加权资产。优势一：风险加权资产覆盖资产负债表中全部资产，没有遗漏任何一类资产的风险敞口，对兴业这类投资资产占比高的银行一样公平；优势二：以风险系数为权重加权，天然考虑了不同资产的风险差异——高风险的资产权重高，低风险的资产权重低，使得指标对不同资产结构的银行具有可比性；优势三：分母（风险加权资产）是客观计算值，不受不良认定主观性的干扰。

**第三环：判断标准。**

指标越大 → 银行用减值准备覆盖其全部风险资产的比例越高 → 风险抵补能力越强。指标越小 → 减值准备不够充分 → 未来需要计提更多减值来抵御风险 → 利润承压。趋势比绝对值更重要——连续3年上升说明银行在"屯粮"，下降说明在"吃老本"。

**第四环：适用范围判断。**
此指标对"贷款外资产占比高"的银行（兴业、平安等持有大量应收款项类投资/非标的银行）诊断价值最大——传统指标会高估这些银行的风险抵补能力（因为只看贷款准备了，漏掉了投资资产的风险）。对贷款占绝对主导的传统银行（如大型国有行），此指标与拨贷比的排名差异相对较小，但仍有修正主观认定偏差的价值。

## A1 — 书中的应用 (Past Application)

### 案例: 2016—2018年9家银行风险加权资产减值准备率对比（c15）

- **问题**: 9家大中型银行（四大国有行 + 四大股份行 + 平安银行），哪家银行用减值准备覆盖全部风险资产的能力最强？传统指标能否准确反映这一能力？
- **方法论的使用**: 作者选取2016、2017、2018年3个年报时点，计算9家银行的风险加权资产减值准备率（资产减值准备÷风险加权资产），获得表6.11数据，做横比+纵比。
- **结论**: 
  - 招行连续3年排名第一，且指标在不断提升——说明风险抵补在持续"加厚"
  - 农行数值虽好，但最近3年在下降——传统指标可能看不出这一点，但趋势恶化是预警信号
  - 建行稳步上升，是国有大行中趋势最健康的
  - 不同银行间差异显著，这一指标比拨备覆盖率和拨贷比更能捕捉到风险抵补的结构性差异
- **结果**: 作者证明了该指标确实能提供拨备覆盖率和拨贷比无法提供的增量信息——尤其是农行（传统指标看起来好的银行），其风险加权资产减值准备率的下降趋势揭示了"家底在消耗"的隐忧。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. **横向对比多家银行的风险抵补能力**：用户问"招行、兴业、民生，谁的风险准备最充足？"→仅比较拨备覆盖率（受不良认定干扰）和拨贷比（忽略风险差异）会给出不同甚至矛盾的排序，需用风险加权资产减值准备率做最终仲裁。
2. **评估投资资产占比高的银行**：用户问"兴业银行的应收投资那么大，它的减值准备够吗？"→这是该指标最有价值的应用场景——传统拨备指标只看贷款，发现不了投资资产的减值缺口。
3. **识别"看起来很安全但实际在吃老本"的银行**：用户发现某银行拨备覆盖率高达200%以上→看它3年的风险加权资产减值准备率趋势。如果覆盖率在高位但该指标持续下降，说明银行在通过"不做实不良认定"维持覆盖率虚高，而非真正在加固风险堤坝。
4. **银行股投资组合构建时的风险抵补排序**：用户问"银行板块里，谁的'安全垫'最厚？"→用风险加权资产减值准备率做横向排序，并结合趋势（上升/下降）做两维筛选。
5. **资产质量拐点的预判**：用户关注某银行的不良生成在加速→看它的风险加权资产减值准备率是否同步提升。如果新生成不良上升但该指标下降，说明银行既没能力也没意愿做足准备，是最差的信号组合。

### 语言信号

- "拨备够不够""风险抵补""减值准备充足吗"
- "拨备覆盖率只能看贷款""除了拨备覆盖率还有什么指标"
- "兴业银行的投资资产减值怎么办"
- "哪家银行的安全垫最厚"
- "风险加权资产减值准备率"

### 与相邻 skill 的区分

- 与 `npl-lifecycle-analysis` 的区别：不良贷款生命周期分析只聚焦于"新生成不良→不良核销"这条流量线，关注的是信用风险的"流入/流出速度"。本skill聚焦于风险抵补的"存量厚度"——银行手里有多少弹药（减值准备）来应对风险资产的全部敞口。前者是风险发生的速度，后者是应对风险的储备。
- 与 `provision-profit-identification` 的区别：信用减值调节利润识别关注的是"银行是否在通过少提减值来美化净利润"。本skill关注的是减值准备累积的"总量是否充足"——即使一家银行计提节奏正常，如果多年风险加权资产减值准备率持续走低，仍然说明风险敞口在扩大而减速跟不上。

## E — 可执行步骤 (Execution)

### 1. 获取数据

- 从年报"财务报表附注"→"资产减值准备"部分获取**资产减值准备余额**（注意是余额而非当期计提，含一般准备+专项准备+特种准备，覆盖贷款减值+投资减值+表外项目减值等全部类别）
- 从年报"资本管理"或"风险管理"章节获取**风险加权资产**总额
- 若年报未直接披露风险加权资产，可从各级资本充足率反推：风险加权资产 = 核心一级资本净额 ÷ 核心一级资本充足率
- 完成标准：至少获取最近1年的数据；最好获取连续3年数据以判断趋势

### 2. 计算风险加权资产减值准备率

- 公式：风险加权资产减值准备率 = 资产减值准备余额 ÷ 风险加权资产 × 100%
- 完成标准：得到数值（通常落在2%-6%区间，招行标杆约5%-6%）
- 判停点：若目标银行未上市或数据完全缺失，提示"数据不可得"，改用拨备覆盖率+拨贷比做替代判断，但标注不确定性

### 3. 与拨备覆盖率和拨贷比交叉验证

- 同时计算目标银行的拨备覆盖率（拨备÷不良贷款）和拨贷比（拨备÷贷款总额）
- 对比三个指标的排序关系：
  - 如果三个指标排序一致 → 银行的资产结构和不良认定都比较干净，结论可信度高
  - 如果拨备覆盖率高但风险加权资产减值准备率低 → 可能是分母问题：不良认定偏低估，或者有大量非贷款资产未被拨备覆盖
  - 如果拨贷比高但风险加权资产减值准备率低 → 贷款资产风险权重高（风控能力差），减值准备相对于风险资产不足
- 完成标准：能用一句话解释"为什么三个指标给出的信号不一致？差异的根源是什么？"

### 4. 做纵向趋势判断（至少3年）

- 用3年数据画趋势线：连续上升/高位震荡/连续下降
- 结合银行同期的营收增长和不良生成趋势做一致性检验：
  - 风险加权资产减值准备率↑ + 营收↑ → 银行在丰年"屯粮"，最佳信号
  - 风险加权资产减值准备率↓ + 营收↓ → 银行在歉年"吃老本"但情有可原
  - 风险加权资产减值准备率↓ + 营收↑ → 银行在增长但风险敞口增长更快，需要警惕
  - 风险加权资产减值准备率↓ + 不良生成↑ → 最差信号，双重恶化
- 完成标准：能判断该银行风险抵补的趋势方向及匹配度

### 5. 综合结论

- 给出该银行风险抵补的三级判定：
  - **厚家底**（绿色）：指标>4%且连续3年上升或稳定 → 减值准备充足，能抵御资产质量的周期性波动
  - **勉强够用**（黄色）：指标在2.5%-4%但趋势不明显或下降 → 关注后续是否能稳住，警惕风险敞口扩大
  - **家底薄**（红色）：指标<2.5%且持续下降 → 减值准备不足，未来需要大量计提，利润将承压
- 完成标准：给出明确评级，并列出支持该评级的3个关键数据点

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 分析非银行金融机构（保险、券商、信托）——风险加权资产是银行业特有监管概念
- 用户明确只需要拨备覆盖率的简单对比，且不涉及"非贷款资产减值""银行间不良认定差异"等深层问题→应先确认用户是否愿意深化分析
- 数据严重残缺（连风险加权资产都无法推算）时强行计算→应标记不确定性而非填补数据
- 在没有行业基准的前提下对单家银行孤立下结论→此指标是相对比率，必须在横向对比中才有意义（本文提供的9家银行2016-2018年数据可作为历史基准锚点）

### 作者在书中警告的失败模式

- **只看不良率就判断资产质量（ce04）**：用户被某银行1.5%以下的低不良率吸引，认为其资产质量好 → 但如果不看风险加权资产减值准备率，可能忽略两个关键：(1) 不良率是被核销和分类修饰过的'时点快照'，不良偏离度可能超过100%；(2) 减值准备相对于全部风险资产可能严重不足。民生银行2017年不良率1.71%，看似与招行1.61%接近，实则不良偏离度135.1% vs 79.5%，意味着民生大量不良被藏在关注类贷款中。仅看不良率+拨备覆盖率会得出'拨备充足'的错误结论，风险加权资产减值准备率则能暴露出减值准备总量相对于全部风险敞口的不足。

- **忽略拨备覆盖率的"分母操纵"陷阱**：银行可以通过少认定不良（把问题贷款留在关注类），使不良贷款余额这个分母变小 → 拨备覆盖率虚高 → 看起来风险抵补充足，但实际上减值准备总量没变，且全部风险资产中尚未暴露的问题远多于认定数。风险加权资产减值准备率用客观的RWA做分母，不受不良认定主观性的干扰。

- **忽略投资资产的减值缺口**：典型如兴业银行在应收款项类投资中放有七八千亿元类贷款资产，传统拨备指标只覆盖贷款部分的减值，用户容易产生"拨备充足"的错觉。风险加权资产减值准备率将所有资产纳入分母，能揭示非贷款资产未被充分覆盖的风险敞口。

### 作者的盲点 / 时代局限

- **非监管指标，缺乏标准化披露**：风险加权资产减值准备率是作者自创指标，银行不直接披露此指标，投资者需自行计算。不同银行对"资产减值准备"的统计口径可能存在细微差异（部分银行的投资减值可能与贷款减值合并披露），影响不同银行间的精确可比性。

- **高级法与权重法计量差异影响分母可比性**：工、农、中、建、交、招行6家使用信用风险内部评级高级法，其他银行使用标准权重法。高级法下，零售占比高、资产质量好的银行可以显著降低风险加权资产 → 分母变小 → 相同减值准备下该指标被抬高。因此跨方法比较时，指标差异中有一小部分来自计量方法差异而非减值准备充足度的真实差异。应以同方法组内比较为主，跨方法比较时标注此因素。

- **数据截止2019年，未覆盖疫情后周期**：本书出版于2021年，核心数据使用2016-2018年。2020年疫情后银行业普遍加大减值计提（监管要求的"前瞻性拨备"），风险加权资产减值准备率可能出现"跳升"→难以区分是真实的风险抵补改善还是监管推动的一次性加厚。使用时应代入最新年报数据重新计算。

- **仅测试了9家大中型银行**：作者的对比样本限于四大国有行+四大股份行+平安银行。对于城商行、农商行等资产结构和风险特征差异巨大的银行，该指标的适用性和基准范围尚未验证，使用时应提高判断阈值（要求更保守的结论）。

- **分子"资产减值准备"的会计处理变化**：2018年后中国银行业实施IFRS 9（预期信用损失模型），减值计提从"已发生损失模型"转为"预期损失模型"，减值准备的统计口径和历史可比性受到了影响。2018年前后的该指标可能出现断点，纵向比较时需要标注准则变更。

### 容易混淆的邻近方法论

- **混淆风险加权资产减值准备率与拨贷比**：两者都是"减值准备÷某个资产分母"。拨贷比分母=贷款总额（1:1权重，忽略风险差异），本指标分母=风险加权资产（按风险权重加权，覆盖全部资产）。拨贷比高只能说明贷款拨备足，本指标高说明全表风险敞口拨备足。一家银行如果拨贷比3%但本指标仅2%，说明它的贷款资产风险权重偏高（可能是大量对公高风险贷款）或有大量投资资产未被拨备充分覆盖。

- **混淆风险加权资产减值准备率与拨备覆盖率**：拨备覆盖率 = 拨备÷不良贷款，分子相同但分母不同。拨备覆盖率受不良认定主观性干扰（分母可操纵），本指标分母为客观计算的RWA。一家银行可以同时做到拨备覆盖率很高（不良认定严格→分母小）和风险加权资产减值准备率很低（RWA增长快→分母大），这恰恰是"表面安全、实际薄弱"的最危险信号。

- **将绝对水平视为"充分"或"不足"的固定标准**：该指标没有监管要求的硬性最低线，4%或6%都没有天然的"安全值"。判断基准必须在横向排序（同一时间多家银行对比谁高谁低）和纵向趋势（同一银行3年走向上升还是下降）两个维度上同时建立。
