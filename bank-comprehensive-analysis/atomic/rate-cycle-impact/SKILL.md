# 加息降息对银行息差的影响推演框架

## R — 原文 (Reading)

> 很多投资者可能想当然地认为，在对称加减息的情况下，银行的净息差不会受到影响。但是，实际情况完全相反，即使是在对称加减息的情况下，银行的净息差也会受到很大的影响。
>
> — 价投谷子地，《看透银行》第4章 4.1.4

> 考虑上面两种不同的组合在不同加息周期下的影响，会出现两种不同的"幸福痛苦曲线"。这两种曲线存在明显的差别，首先同业业务在高息周期整体处于"痛苦区"，而在低息周期整体处于"幸福区"；而存贷业务正好相反。另外一个差别是，同业业务在加息周期中存在一个"非常痛苦区"，这是期限错配造成的，短期负债利率重定价周期明显短于长期资产的重定价，而这一"非常痛苦区"只有在加息速率减慢或者持平后才会结束。
>
> — 价投谷子地，《看透银行》第4章 4.1.5

> 无论采用何种加息策略，相对来说A银行（存贷为主、活期占比高）的息差表现都会优于B银行（同业为主、活期占比低）。
>
> — 价投谷子地，《看透银行》第4章 4.1.5

## I — 方法论骨架 (Interpretation)

本框架的核心是：**利率周期变化对银行净息差的影响，取决于两条轴——加息路径（3种模式）× 银行类型（2种资产负债结构），构成一个 3×2 的推演矩阵。**

### 第一轴：三种加息路径

| 路径 | 机制 | 影响范围 |
|------|------|----------|
| **加基准利率（对称/非对称）** | 央行上调存贷款基准利率。对称=各档期存贷同幅，非对称=存贷幅度不同。活期通常不跟随上调 | 主要影响存贷业务：贷款利率重定价、定期存款到期重定价 |
| **加央行拆借利率** | 央行通过MLF/SLF等货币工具提高金融机构从央行获取负债的成本 | 直接影响同业拆借市场利率（Shibor），通过LPR（=MLF+加点）间接传导至贷款定价 |
| **数量型货币工具** | 央行通过正回购、提高准备金率等收紧流动性 | 推高市场利率，效果类似加拆借利率但更间接 |

### 第二轴：两种银行类型

| 特征 | A银行：存贷型 | B银行：同业型 |
|------|--------------|--------------|
| 核心业务模式 | 存款配对贷款 | 同业负债搭配同业资产+应收投资 |
| 活期占比 | 高（如招行活期>60%） | 低 |
| 负债结构 | 存款为主，同业负债为辅 | 同业负债占比高（如兴业>31%） |
| 期限错配程度 | 低，活期不重定价 | 高，短借长贷依赖期限错配赚息差 |
| 代表银行 | 四大行、招商银行 | 兴业银行、部分股份行 |

### 3×2 推演矩阵：三种加息路径下两类银行息差变化

| 加息路径 | A银行（存贷型，活期占比高） | B银行（同业型，活期占比低） |
|----------|---------------------------|---------------------------|
| **加基准利率（对称）** | **短期**：息差有一定增加（短期贷款重定价）
**中期**：息差明显增加（长期贷款重定价，活期不跟涨）
→ **幸福区** | **短期**：息差基本不变（短期贷款+短期同业负债同时重定价，互相抵消）
**中期**：息差有一定增加（长期贷款重定价）
→ 温和幸福 |
| **加基准利率（非对称，存款涨幅更激进）** | **短期**：息差基本不变或略微增长
**中期**：息差有一定增加
→ 幸福减弱 | **短期**：息差略微下降（存款成本上升+同业负债成本上升）
**中期**：息差基本不变或略微下降
→ 痛苦边缘 |
| **加央行拆借利率（MLF/正回购）** | **短期**：息差基本不变或略微增长（短期贷款跟市场利率重定价，活期和长期贷款不变）
**中期**：基本不变或略微增长
→ 温和幸福 | **短期**：息差略微下降（短期同业负债立即重定价↑）
**中期**：息差有一定下降（未脱离"非常痛苦区"，长资产利率滞后于短负债利率）
→ **非常痛苦区** |

**降息周期则完全相反**：存贷型银行处于"痛苦区"（贷款利率下降快于存款成本），同业型银行处于"幸福区"（负债成本下降快于资产收益下降）。

### 两条"幸福痛苦曲线"的本质差异

```
存贷型银行的幸福痛苦曲线：
  加息周期 ──幸福区──（无非常痛苦区，因活期不动）
  降息周期 ──痛苦区──

同业型银行的幸福痛苦曲线：
  加息周期 ──痛苦区──>非常痛苦区（期限错配放大）──
  降息周期 ──幸福区──
```

**关键机制**：
1. **活期存款是"息差稳定器"**：活期利率几乎不跟随任何加息模式变动，活期占比越高，银行在加息中的受益越大，在降息中的受损越小。
2. **期限错配是"痛苦放大器"**：同业型银行短借长贷，加息时短期负债立即重定价（成本飙升），而长期资产利率调整严重滞后，二者剪刀差快速压缩息差。
3. **"非常痛苦区"的持续时间**：只有当市场利率上升趋势明确放缓或结束，长端资产完成重定价后，同业型银行的息差压力才会缓解。在此之前，"资产配置是一个连续的过程"——每一笔新做的同业业务依然处于错配挤压中。

## A1 — 书中的应用 (Past Application)

### 案例：对称加息对两类银行净息差影响的量化模拟（c07）

- **问题**：对称加息（贷款各档期+25bp，存款各档期+25bp）对活期占比不同的两家银行，净息差影响差多少？
- **方法论的使用**：作者构建两个假设银行的资产负债模型——银行A活期占比高，银行B活期占比低，各自只有存款、贷款和自有净资产三类。在对称加息场景下，活期利率不跟随上调（这是现实中的常态），自有净资产不产生利息也无影响。计算加息前后的净利息收入变化。
- **量化结果**：
  - 银行A（活期占比高）：净利息收入增加17.5亿元，净息差扩张 **0.175个百分点**
  - 银行B（活期占比低）：净利息收入增加12.5亿元，净息差扩张 **0.125个百分点**
  - 两者差距：活期占比高的银行在同等加息下，息差扩张幅度多出 **40%**
- **延伸验证**：在非对称加息场景（存款加息幅度更大，且活期加息5bp）下，银行A净利息收入仅增7亿元（息差+7bp），银行B仅增3亿元（息差+3bp）。活期一旦参与加息，息差扩张严重受限。
- **结论**：对称加息≠息差不变化。净息差扩张的根本驱动力是"活期存款不跟随加息"，活期占比是决定银行在加息周期中受益程度的核心变量。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill？

1. **央行调整利率时判断银行持仓**：用户问"央行今天加息了，我该重仓招行还是兴业？"→需要用 3×2 矩阵判断具体加息模式，匹配银行类型给出推演。
2. **降息周期中银行筛选**：用户问"LPR连续下调，哪些银行抗跌？"→存贷型银行痛苦区，同业型银行幸福区，需要结合最新利率路径做反向推演。
3. **利率拐点前的前瞻布局**：用户问"利率见底了吗？要不要提前布局存贷型银行？"→需要判断加息模式预期，用幸福痛苦曲线做前瞻推演。
4. **特定银行在已知加息路径下的压力测试**：用户问"如果MLF上调50bp，兴业银行息差会降多少？"→需要判断该银行的同业负债占比和期限错配程度，估计"非常痛苦区"的深度。
5. **比较两家不同类型银行在同一利率环境下的分化**：用户问"为什么这次加息招行涨兴业跌？"→需要用两类银行的不同幸福痛苦曲线解释分化机制。

### 语言信号

- "加息/降息对XX银行影响多大"
- "利率上调，该买哪家银行"
- "MLF/LPR变化后银行息差怎么走"
- "加息利好所有银行吗"
- "降息周期里活期占比高的银行会怎样"
- "同业型银行和存贷型银行在利率变化中谁更受伤"
- "非常痛苦区"是什么意思

### 与相邻 skill 的区分

- 与 `nim-decomposition` 的区别：净息差拆解框架做的是**静态快照**（给定一个年报时点，息差从资产端和负债端分别来自哪），本 skill 做的是**动态推演**（给定一个利率变化场景，息差向哪个方向变动、幅度多大）。两者互补：先用 nim-decomposition 拆现状，再用本 skill 判断未来走向。
- 与 `liability-cost-evaluation` 的区别：负债成本评估框架聚焦**负债端的单维度深度分析**（存款活期占比、同业负债结构、负债成本趋势），本 skill 同时考虑资产端和负债端对利率的重定价响应差异，核心输出是"息差方向+幅度"的跨周期推演。

## E — 可执行步骤 (Execution)

### 步骤 1：识别利率变化的模式

- **确认本次利率变动的类型**：
  - 是基准利率调整（对称/非对称）？→ 进入步骤 2a
  - 是央行拆借利率/MLF调整？→ 进入步骤 2b
  - 是数量型工具（降准/正回购等）？→ 属于间接影响，效果类似拆借利率但更弱，进入步骤 2b
- **确认变动的方向**：加息 or 降息？（降息周期所有结论反向）
- 完成标准：能用一句话描述"本次加息路径是：对称加基准利率25bp" 或 "通过MLF加息10bp"
- 判停条件：若无法区分加息模式，不要笼统下结论，先要求用户提供具体利率政策信息。

### 步骤 2：目标银行归类

- **2a. 对于基准利率（对称/非对称）场景，重点分析存贷款业务的息差响应**：
  - 获取目标银行活期存款占比（年报"存款结构"章节）
  - 获取短期贷款（一年内到期或重定价）占比和长期贷款占比
  - 推断：活期占比越高→在对称加息中受益越大（活期不重定价）；短期贷款占比越高→息差响应越快
  - 完成标准：能说出"该银行活期占比X%，短期贷款占比Y%，预计对称加息对称息差扩张约Z bp"
- **2b. 对于央行拆借利率/MLF场景，重点分析同业业务的息差响应**：
  - 获取目标银行同业负债（拆入+同业存放+同业存单）占总负债的比例
  - 获取同业资产（拆出+买入返售+应收款项类投资）占总生息资产的比例
  - 判断期限错配程度：短期同业负债 vs 长期同业资产（应收投资/非标）的久期差距
  - 完成标准：能说出"该银行同业负债占比X%，同业资产久期约Y年，期限错配程度为____，预计进入非常痛苦区的深度为____"
- 判停条件：若无法获取同业负债分项数据，至少应在年报附注中找到"同业及其他金融机构存放款项"的利率水平和到期期限结构。

### 步骤 3：套入 3×2 矩阵做推演

- 将识别出的加息模式（行）和银行类型（列），在 I 节的 3×2 矩阵中定位交叉单元格
- 区分"短期影响"（3个月内，短期资产/负债重定价主导）和"中期影响"（6-12个月，长期资产/负债逐步完成重定价）
- 特别关注：如果是央行拆借利率加息且目标银行为同业型银行，需要额外判断是否处于"非常痛苦区"——即市场利率上升趋势是否仍在持续。
- 完成标准：能用矩阵语言输出推演结论，格式为"在____加息路径下，该银行短期息差____（方向+幅度估算），中期息差____（方向+幅度估算）"

### 步骤 4：反向推演（降息场景）

- 若当前环境为降息，将步骤 1-3 的所有结论取反方向
- 特别注意：降息时，同业型银行从"非常痛苦区"进入"幸福区"的节奏同样受期限错配影响——短期负债立即享受降息，中期资产收益滞后下降，息差阶段性走宽。这是判断降息周期中反向投资机会的关键窗口。
- 完成标准：降息场景推演覆盖短期和中期，且说明两条幸福痛苦曲线在降息区的位置。

### 步骤 5：量化估算（如数据充分）

- 参考 c07 案例的简化模型方法：以目标银行年报的生息资产/计息负债数额和收益率/成本率为基准，假设利率变动的具体bp数值，计算净利息收入的边际变化。
- 核心公式：
  - 净利息收入变化 ≈ Σ（每类生息资产 × 该类资产利率变动） - Σ（每类计息负债 × 该类负债利率变动）
  - 净息差变化 = 净利息收入变化 ÷ 总生息资产
- 重点变量：
  - 活期存款：利率变动通常 = 0（不跟随加息）
  - 短期贷款：即时重定价，利率变动 = 基准利率变动 × 1
  - 长期贷款（按揭等）：滞后重定价，利率变动约在半年～一年后生效
  - 同业负债（短期）：即时重定价，利率变动 ≈ MLF/Shibor变动 × 1
  - 同业资产/应收投资（长期）：滞后重定价
- 完成标准：给出净息差变化的 bp 估算区间

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 仅做宏观经济判断（"加息利好股市""降息刺激经济"）而不涉及具体银行资产负债结构分析——这是宏观研究员的工作，不是本 skill 的用途
- 不区分加息路径（对称/非对称/央行拆借）就笼统断言"加息=银行利好"——这种粗糙判断正是 ce02 反例所警示的
- 不区分银行类型就给出统一结论——不同银行在同一加息场景下可能表现相反
- 用户没有提供或无法获取目标银行资产负债结构数据时强行推演——没有数据支撑的推演等于猜
- 分析非银行机构（保险公司、券商）受利率影响——它们的业务模型和资产负债结构完全不同，不适用于此框架

### 作者在书中警告的失败模式

- **对称加减息误判（ce02）**：最经典的认知陷阱——投资者想当然认为各档期存贷款利率同步升降时息差不变。实际上因为活期存款利率几乎从不跟随基准利率调整，对称加息时活期占比高的银行净息差仍扩张。"忽略负债结构的非对称性"是银行利率敏感性分析中最危险的错误。

- **误将利率市场化等同于净息差必然收窄（ce03）**：投资者以日本和中国台湾利率市场化后息差崩溃为先例，得出中国银行业必然受重创的结论。但日本/台湾的息差崩溃根因是经济衰退而非利率市场化本身——利率是资金的价格，最终由实体经济对资金的供求决定。美国利率市场化后净息差长期维持合理区间就是反证。在预测加息/降息对中国银行的影响时，不能只看利率工具本身，必须结合名义GDP增速和实体经济的资金需求。

- **利率管制下储户被系统性剥夺（ce12）**：在1990-2016年近30年利率管制时期，一年期存款利率仅2%-4%，而名义GDP年化增速约15%。储户的实际收益远低于社会无风险利率，实质是将财富持续转移给了贷款买房/经营的人。这一机制提示：在分析利率政策对银行的影响时，不能把"存款利率"等同于"储户获得的价格"——管制利率下的存款成本是被人为压低的。

### 作者的盲点 / 时代局限

- **数据截止2019年，LPR新机制运行尚短**：本书出版于2021年，对LPR定价机制（=MLF+加点）的分析仅基于2019年8月新机制推出后的初期观察。2020-2025年间LPR多次单边下调（MLF降但存款基准利率不动），出现了非对称降息的特殊模式，这种"贷款降、存款不降"对银行息差的挤压模式超出了书中假设的对称/非对称框架，需要结合最新LPR改革进展调整推演。

- **假设活期利率永不调整可能不再成立**：书中框架的核心支柱是"活期存款不跟随利率调整"。但随着利率市场化深化和存款利率自律机制的形成（2022-2023年多次调整存款利率上限），存款利率也出现了"跟随MLF/LPR"下调的趋势。活期存款的"零利率刚性"在长期低利率环境下可能被打破，届时存贷型银行在加息中的"幸福区"优势将被削弱。

- **未考虑利率变化对资产质量的二阶效应**：加息不仅改变息差，还会推高借款人的还款负担→不良率可能上升。本书在4.1.4-4.1.5中仅讨论息差的一阶效应，对加息引发信用成本上升的二阶效应未做分析。在高杠杆环境下（如房产/地方债），快速加息可能引发的不良反弹会通过信用减值损失反噬息差收益。

- **未覆盖负利率或零利率极限场景**：若利率降至零附近（如日本式长期低利率），传统"降息→息差收窄→银行受伤"的线性推演将失效。在零利率下，存款成本已无下降空间（下限为0），而贷款收益率仍可能持续走低，息差面临单向压缩，两类银行的"幸福痛苦曲线"都将失效。

### 容易混淆的邻近方法论

- **用ROE而非息差方向评价利率影响**：加息周期中，有些银行虽然息差扩张（受益），但如果同时遭遇不良率上升（受损），ROE可能不升反降。本 skill 仅推演息差方向，不直接评估最终利润——还需要叠加资产质量分析（如 npl-lifecycle-analysis）才能完整判断。

- **混淆LPR调整与基准利率调整**：2019年LPR新机制后，LPR=MLF+加点。MLF调整直接影响LPR→贷款利率重定价，但对存款利率无直接影响。这是一种"非对称的利率传导"，不同于传统的"基准利率对称加息"。在套用3×2矩阵时，应把"MLF加息"归入"加央行拆借利率"路径，而非"加基准利率"路径。

- **简单线性外推**：不能因为某银行在上一次加息中息差扩张了10bp，就断言下一次同等加息也会扩张10bp。银行会主动调整资产负债结构（如提前拉长同业负债久期、调整贷款客户结构）来应对预期中的利率变化，使得同样的加息路径产生不同的效果。推演结论必须是有方向性区间的、条件性的，而非精确的。
