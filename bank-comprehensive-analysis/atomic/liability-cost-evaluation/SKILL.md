# 负债成本双维度评估 + AUM导向分析

## R — 原文 (Reading)

> 考察一家银行的负债业务是优秀还是平庸，最重要的指标就是负债成本。负债成本包含两部分：一个是直接成本，一个是间接成本。直接成本是指银行全部计息负债的综合利率，用综合利率乘以计息负债就可以得到一家银行的利息支出。间接成本是指银行为了获取足够多的负债需要投入多少营运费用，包括网点租金、人工薪酬、设备采购费等。
>
> — 价投谷子地, 第1章 1.1节

> AUM导向，就是放弃过去对银行员工的存款考核指标，转而用AUM指标……放弃追逐存款反而成就了招行超高的活期存款占比。招行的活期存款，主要来源于两大类：第一，主动性活期存款，主要来源于高净值客户……第二，被动性活期存款，主要来源于高收益理财产品的集合期和间歇期。
>
> — 价投谷子地, 第1章 1.2.2节

> 招行的目标从来只有一个：追赶客户的脚步，打造最佳客户体验银行，而活期存款不过是在这个过程中客户对招行的馈赠。
>
> — 价投谷子地, 第1章 1.2.2节

## I — 方法论骨架 (Interpretation)

评估银行的负债能力，不能只看存款总量的增长速度，必须从**直接成本 + 间接成本**两个维度入手。

**直接成本（计息负债综合利率）** 是银行支付给储户和债权人利息的显性成本。决定直接成本高低的不是存款总量，而是**活期占比**——活期存款利率接近零，活期占比越高的银行，直接存款成本越低。仅靠几十个基点的成本差异，乘以数万亿的存款规模，就是数百亿元的利息支出差距。

**间接成本（营运费用分摊）** 是银行为了获取负债而在网点、人工、系统等方面投入的隐性成本。四大行凭借遍布全国的网点获得了最低的直接成本（但因网点多，间接成本也高于以对公为主的股份行）。而招行夹在中间：直接成本略高于四大行、远低于其他股份行，但间接成本最高——因为服务高端零售客群需要更高的运营投入。

**AUM导向的反直觉逻辑** 是本方法论的核心洞察。常规思维是"存款立行"——越考核存款、越给高息揽储，存款越多。但事实恰恰相反：
1. 越追存款 → 存款中定期占比越高 → 负债成本越贵
2. 不追存款（AUM考核）→ 给客户推荐理财和基金 → 客户体验更好 → 更多资产转入银行 → 活期沉淀（理财募集期+高净值流动性需求）反而更多 → 负债成本更低

活期存款本质上不是"拉"来的，而是在服务客户过程中自然沉淀的。AUM导向将银行从"利率战"拉回到"服务战"，这种体系其他银行短期无法复制（因为切换考核体系会导致短期存款流失，管理层承受不住压力）。

因此，负债端评估的完整框架是：**先看直接成本（活期占比是核心驱动）→ 再看间接成本（判断是良性投入还是低效扩张）→ 最后穿透负债结构的稳定性（存款占比 vs 同业负债占比）→ 最终判断这家银行负债优势是否可持续**。

## A1 — 书中的应用 (Past Application)

### 案例 1: 9家大中型银行2017年负债成本横向对比

- **问题**: 在9家典型银行（工、农、中、建、招及浦发、民生、兴业、平安）中，如何客观地判断谁的负债能力最强？仅看谁的存款多或谁的增长快远远不够。
- **方法论的使用**: 作者提取9家银行2017年年报数据，从直接成本和间接成本两个维度逐项拆解。直接成本看计息负债综合利率，间接成本用营运费用除以计息负债进行估算（假定负债业务和资产业务各占营运费用的一半）。然后按负债能力将9家银行分为两大阵营。
- **结论**: 四大行直接成本1.53%～1.69%（网点优势），间接成本0.41%～0.50%。"浦民兴平"四家股份行直接成本2.53%～2.81%（以对公为主、活期占比低），间接成本反而低于四大行（因不养大量零售网点）。招行最特殊：直接成本介于两者之间，间接成本全表最高——因为"服务高端人群"的战略定位导致运营投入大。
- **结果**: 此案例建立了"直接成本+间接成本"双维度评估的标准框架，使下游所有银行负债分析有了可比性的基准。

### 案例 2: 招商银行AUM导向零售体系如何成就低成本负债

- **问题**: 招行网点数量仅为四大行的约1/10，凭什么做到了1/6的零售存款，且零售活期占比高达72.64%？按常识，客户不可能"傻到"不存余额宝却存活期。
- **方法论的使用**: 作者深入分析招行2017年年报的客户AUM分布数据（零售AUM高达6万亿元，是零售存款的4.61倍；不到2%的高净值客户占有82%的AUM），再结合AUM考核体系的机制设计（不考核存款→不推高息定期→理财+基金为主→客户体验好→钱转入越多→活期沉淀越多），完整揭示了活期存款的两大来源。
- **结论**: 活期来自两部分：（1）高净值客户的主动性流动性需求（T+0产品每日限额1万，无法满足大户流动性）；（2）理财产品的募集期和产品间歇期的被动沉淀（想抢到高收益产品必须提前存入活期）。AUM导向把招行从"利率战"拉回"服务战"，活期只是服务好客户后的"馈赠"。
- **结果**: 招行的负债成本优势成为其最高的"护城河"之一，其他银行因短期存款考核压力无法放弃传统的存款指标，无法复制AUM体系。这解释了为什么招行在银行板块中长期保持净息差领先。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **横向对比多家银行的负债竞争力**: 用户手上有几家银行（如招行、建行、兴业、平安）的存款数据，想判断谁的负债端更优秀、成本更低、结构更稳定。
2. **判断银行存款增长的"含金量"**: 用户发现某银行存款增速很快（如兴业），想知道这是真的竞争力提升还是在用高息产品"买存款"。需要区分"数量增长"和"质量改善"。
3. **分析银行推出新产品（如高息活期）的利弊**: 用户看到某银行新产品大幅拉动存款增长，需要判断这究竟是在强化还是在破坏银行的负债成本优势。
4. **理解招行的负债成本护城河**: 用户问"招行比兴业/平安到底好在哪里"、"为什么招行能做到这么低的负债成本"、"AUM考核体系是什么"。
5. **评估利率环境变化对不同银行负债端的影响**: 利率市场化推进、存款竞争加剧的趋势下，哪些银行的负债结构更抗压？

### 语言信号（用户的话里出现这些就应激活）

- "这家银行的负债成本怎么样"
- "A银行和B银行的存款质量对比"
- "为什么XX银行存款增速快但成本高"
- "活期占比对银行负债成本的影响"
- "AUM考核体系是什么意思"
- "招行的活期占比为什么这么高"
- "这个新产品（靠档计息/高息活期）对银行是好事吗"
- "银行存款多了但成本也涨了，是好是坏"

### 与相邻 skill 的区分

- 与 `nim-decomposition`（净息差拆解）的区别：负债成本评估只聚焦负债端，不涉及资产端的收益率分析。本 skill 是净息差拆解的"左半部分"前置分析。
- 与 `asset-quality-analysis`（资产质量分析）的区别：本 skill 分析钱的来源（负债），后者分析钱的去向（资产）。两者可以组合使用，但触发信号完全不同。
- 与 `cmb-moat-analysis`（招行护城河分析）的区别：本 skill 聚焦AUM体系对负债成本的影响机制，是招行护城河的一个子模块；后者覆盖战略、风控、客群等更广维度。

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行：

1. **收集目标银行的负债端核心数据**
   - 从年报或财报中提取：存款总额、存款增速（近期+长周期）、活期存款占比（对公活期占比+零售活期占比）、计息负债综合利率、营运费用总额。
   - 完成标准：已获得至少两家银行（目标银行+至少一家对标银行）的上述数据，形成对比表格。

2. **直接成本分析：计算并对比计息负债综合利率**
   - 计算每家银行的计息负债综合利率 = 利息支出 ÷ 计息负债平均余额。
   - 重点分析活期占比差异对直接成本的驱动作用：谁的活期占比更高 → 谁的存款成本更低 → 换算为绝对利息支出差异（存款总额 × 成本差值基点）。
   - 判停条件：如果目标银行的活期占比显著低于对标银行（差10个百分点以上），直接判定负债竞争力弱，无需继续深度分析AUM体系。

3. **间接成本分析：计算并对比营运费用分摊**
   - 估算间接负债成本 = 营运费用 ÷ 计息负债（简化估算可取营运费用×50%÷计息负债，假定负债和资产业务各摊一半营运费用）。
   - 判断间接成本高低的原因：是高零售投入导致的（良性，如招行），还是低效网点扩张导致的（恶性）。
   - 完成标准：能说出目标银行直接成本+间接成本的总负债成本，并定位其在大行/股份行/招行三大阵营中的位置。

4. **AUM体系穿透（仅适用于零售型银行）**
   - 查看银行是否披露AUM数据。如果公布AUM：计算AUM/零售存款倍数（招行约4倍，此倍数越高说明客户资产在非存款产品中的配置越多→服务能力强→活期沉淀潜力大）。
   - 分析存款考核体系：年报/管理层表述中是否提及AUM考核或"平衡计分卡"替代传统存款考核。
   - 判断AUM体系的反直觉效应是否正在发生：活期占比是否持续提升+存款成本是否稳定或下降+客户投诉/流失是否在降低。
   - 完成标准：能回答"这家银行的低负债成本是否可持续"——如果来自AUM体系，可持续性强；如果来自高息揽储，不可持续。

5. **综合评估与风险提示**
   - 输出结论：目标银行负债端竞争力的综合评价（优秀/良好/一般/差），以及核心驱动因素。
   - 标注风险点：存款成本是否有突变上升、是否有新产品（靠档计息等）正在推高活期成本、同业负债占比是否过高影响负债稳定性。
    - 完成标准：输出完整的负债成本评估报告，包含数据支撑和风险提示。

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **单纯比较存款总量或增速，不涉及成本分析** — 存款规模只是"数量"维度，本 skill 的核心是"数量+质量"双维度。如果用户只想知道"哪家银行存款最多"或"谁增长最快"，不需要激活此 skill。
- **分析非银行业企业** — 负债成本的双维度框架（计息负债利率+营运费用分摊）专为银行设计，不适用于工商企业。
- **评估贷款端或资产端** — 本 skill 只覆盖负债端。如果用户的问题是"这家银行贷款利率高不高"或"资产质量如何"，应激活对应的资产端或资产质量 skill。
- **只看一个季度的数据变化** — 负债成本评估需要看较长周期（至少年度）的趋势，单个季度的波动受季节性和会计处理影响较大，不足以做出可靠判断。

### 作者在书中警告的失败模式

- **农行"活利丰"自残案例（ce15）**：农业银行2018年推出"活利丰"靠档计息产品——对活期存款按定期利率计息，短期内存款数量增加，但活期存款成本从原本近乎为零猛增39个基点（接近翻倍），相当于全部存款成本提升12个基点。以农行18万亿存款规模计算，多付了210亿元利息。结果是：牺牲了四大行最核心的负债成本优势（原本可以靠网点免费吸收活期），换来的存款增量在财务上得不偿失。2019年农行股价在小牛市中原地不动，验证了负债成本恶化对股东价值的侵蚀。
- **高息揽储的"数量vs质量"陷阱（c02）**：兴业银行存款增速在9家银行中名列前茅，但因活期占比低、存款成本比招行高62个基点，仅存款利息就多付出186亿元。存款增速高不等于吸储能力强——这种"用高利率买来的存款"，增长的只是负债表的"虚胖"而非竞争力。

### 作者的盲点 / 时代局限

- **作者是招行的长期重仓持有者（持仓超10年）**，全书以AUM体系作为最优解，可能低估了非AUM路径的负债策略在特定条件下的有效性（如邮储银行的县域存款独占优势、宁波银行的小微企业结算沉淀）。
- **数据截止于2018-2019年**：2020年后LPR持续下调、存款利率自律机制改革（2021年后存款利率上限被限制）、理财净值化转型（2022年理财破净潮）等因素可能弱化了AUM体系中"理财产品募集期沉淀活期"这一机制的效力。
- **对互联网银行/数字银行的存款创新缺乏讨论**：微信支付、支付宝等场景嵌入带来的支付结算沉淀资金，可能以另一种方式实现"不追存款而得存款"的效果，独立于传统AUM体系。

### 容易混淆的邻近方法论

- 与 **存款保险制度** 混淆：存款保险讨论的是储户资金安全，本 skill 讨论的是银行获取负债的效率。
- 与 **利率市场化影响评估** 混淆：利率市场化影响银行的全盘利差环境，本 skill 分析微观层面某家银行的负债结构具体优劣势。
- 与 **流动性风险评估** 混淆：流动性风险关注负债的稳定性和期限匹配（存款vs同业负债、短期vs长期），本 skill 关注负债的获取成本。两者相关但不相同——低成本负债未必高稳定（如同业负债成本可能低于高息存款，但稳定性远不如存款）。
