# RORWA评估框架与内生性增长判断法则

## R — 原文 (Reading)

> 对于银行来说，在未来严格监管资本消耗的情况下，如何利用最少的风险资产赚取最多的利润，就成为衡量银行优劣的重要指标。这一指标就是风险加权资产收益率（RORWA），其计算公式为：RORWA=净利润÷风险加权资产。
>
> — 价投谷子地，《看透银行》8.3.2节

> 所谓内生性增长，就是银行在不依赖外部融资的情况下，完全依靠净利润留存来支撑风险资产的增长。只有具备内生性增长的银行才能持续给股东以现金回报，而那些不具备这一能力的银行，基本上隔一段时间就需要进行股权融资。如果采用配股的形式，那么就相当于从股东手中圈钱；如果是定向增发，也会摊薄股东手中的权益。
>
> — 价投谷子地，《看透银行》8.3.3节

> 内生性增长的底线公式——为了保持核充率不变，RORWA需要满足：R=(H×Ta)/(1-p)。其中H为核充率，Ta为风险资产增速，p为分红率。R和核充率、风险资产增速成正比，和(1-分红率)成反比。
>
> — 价投谷子地，《看透银行》8.3.4节

> 可以毫不客气地说，平安银行几乎没有为股东创造任何价值，只是在不停地"吸血"。平安银行是6家中唯一的核心资本充足率2018H1数据低于2014H2数据的银行，累积4年的现金分红还抵不过一次融资。
>
> — 价投谷子地，《看透银行》8.3.3节

## I — 方法论骨架 (Interpretation)

RORWA评估框架与内生性增长判断法则是作者构建的**银行筛选"黄金法则"**，核心逻辑链为：

**第一环：RORWA是评估银行优劣的最佳单一指标。**
传统指标各有缺陷——ROE被杠杆倍数扭曲（杠杆越高ROE越高，而非经营效率越高），ROA绝对值太小不易区分（仅1%左右）。RORWA直接连接"盈利"和"监管约束（资本充足率）"，衡量的是银行用单位风险资产创造净利润的能力。RORWA=净利润÷风险加权资产，同一家银行在相同资产规模下，风险加权资产越低→RORWA越高→使用监管资本越"省"。在巴塞尔协议Ⅲ持续压低杠杆的时代背景下，RORWA比ROE和ROA更精准地捕捉到银行竞争力的分化。

**第二环：RORWA决定内生性增长能力。**
内生性增长是一种银行经营状态——不依赖外部股权融资（增发、配股、可转债转股），完全靠自身净利润留存支撑风险资产增长，并持续提升对股东的现金回报。内生性增长有三个判别标准：
1. 不融资前提下，核心一级资本充足率（核充率）能否振荡攀升
2. 累计净派现能力（分红金额-融资金额）是否为正
3. RORWA是否满足底线公式：R=(H×Ta)/(1-p)

**第三环：底线公式是将RORWA与内生性增长量化的数学桥梁。**
核心一级资本充足率的分子（核心一级资本）中，未分配利润来自净利润扣除分红后的留存。因此RORWA中的"净利润"与核充率存在直接函数关系。给定银行的风险资产增速Ta、核充率H、分红率p，可以精确算出维持核充率不下降所需的最低RORWA。实际RORWA高于底线→核充率上升→内生性增长无虞；实际RORWA低于底线→核充率必然下降→银行迟早需要外部融资。

**第四环：RORWA贡献度分解揭示竞争力来源。**
RORWA差异的背后是银行竞争能力的具象化。将RORWA分解为三项贡献度：
- **扣除信用成本后净利息收入贡献度** = (净利息收入-信用减值损失)÷风险加权资产。核心体现负债成本控制能力——低负债成本→无需高风险定价→低信用成本。
- **手续费和佣金收入贡献度** = 手续费收入÷风险加权资产。体现客户黏性和中间业务服务能力。
- **费用支出贡献度** = 业务及管理费÷风险加权资产。体现营运投入力度（金融科技、人员薪酬）。

## A1 — 书中的应用 (Past Application)

### 案例 1: 2019年9家银行RORWA横向对比——招行一骑绝尘（c17）

- **问题**: 9家大中型银行（工、农、中、建、交 + 招、浦、民、兴、平），哪家银行单位风险资产的盈利能力最强？
- **方法论的使用**: 作者使用2016-2019年年报数据计算9家银行的RORWA。招行2019年RORWA高达2.40%，是最低银行（平安）的2倍多。9家中只有招行连续4年RORWA上升，其他银行普遍下降（农行掉16bp，工行掉12bp）。作者进一步做RORWA贡献度分解（表8.6），发现差距主要来自两项：扣除信用成本后净利息收入贡献度——招行2.90%，民生仅0.69%；手续费和佣金贡献度——招行约第二名的2倍。
- **结论**: RORWA是比ROA更敏感的核心指标。招行2.40%意味着用等量风险资产，招行创造的净利润是平安银行的2倍以上。差距根源是：低负债成本→扣除信用成本后净利息收入贡献度遥遥领先 + 零售客户黏性→手续费收入远超同业。
- **结果**: 招行是唯一一家不依赖融资就能保持核充率持续攀升、同时分红能力持续增长的银行。RORWA连续4年上升是招行"轻型银行"战略成功的最强证据。

### 案例 2: 招商银行vs民生银行——RORWA底线公式的定量验证（c20）

- **问题**: 如何精确判断某家银行是否需要外部融资？能否用一个公式"提前预判"？
- **方法论的使用**: 作者代入招行和民生2019年年报数据验证底线公式。招行：核充率11.95%，风险资产增速约9%，分红率约33%→所需RORWA底线=1.59%，实际RORWA=2.40%（超额达标）。民生：核充率10.12%，风险资产增速约10%，分红率约33%→所需RORWA底线=1.26%，实际RORWA=1.05%（不达标）。
- **结论**: 公式预测与实际报表完全一致——民生2019年核充率同比下降。投资者可以用R=(H×Ta)/(1-p)一次性判断目标银行是否会"伸手要钱"。只有实际RORWA持续高于底线的银行，才值得长期持有。
- **结果**: 该公式将内生性增长的定性判断升级为定量计算，使投资者能像作者一样，在银行发布再融资公告前就预判到融资需求。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. **A股银行板块整体筛选**：用户问"银行板块里哪几家值得长期持有？"→计算RORWA做横向排序，筛选RORWA>1.5%且趋势上升的银行，再用底线公式逐个验证内生性增长。
2. **单一银行的投资决策**：用户问"兴业银行现在能买吗？它需要融资吗？"→用RORWA底线公式代入当前数据，预判核充率走向和融资需求。
3. **再融资事件的解读**：用户问"XX银行又要增发了，是好是坏？"→计算该银行过去3年RORWA与底线的差距，判断是"被迫融资"（价值毁灭信号）还是"战略性融资"。
4. **分红可持续性的判断**：用户问"XX银行的分红能维持30%吗？"→底线公式直接回答了：给定RORWA和风险资产增速，能承受的最大分红率是多少。如果RORWA撑不住当前分红率，分红迟早会被砍。
5. **两家银行的深度对比**：用户问"招行和民生哪个更有投资价值？"→不能只看PE/PB，要用RORWA贡献度分解（表8.6模式）做三项对比，找到差距的结构性来源。
6. **识别"伪增长"银行**：用户被高净利润增速或高净息差吸引→用RORWA检验：如果RORWA长期低于1.2%且趋势下降，说明银行在靠规模扩张而非效率提升，在巴Ⅲ环境下是不可持续的。

### 与相邻 skill 的区分

- 与 `nim-decomposition` 的区别：净息差拆解框架聚焦于净利息收入这条线，拆到"扣除信用成本后净利息收入贡献度"为止。本skill在此基础上继续拆解——加上手续费贡献度、费用支出贡献度，最终汇到RORWA，并连接内生性增长判断。净息差拆解是RORWA分析的"子模块"。
- 与 `liability-cost-evaluation` 的区别：负债成本评估只拆负债端，本skill的RORWA贡献度分解说明负债成本优势才是RORWA领先的根本原因，但本skill的范围远超出负债端。
- 与 `provision-profit-identification` 的区别：信用减值调节利润识别只关注"净利润是否被粉饰"，本skill直接用"扣除信用成本后净利息收入"绕过了这个陷阱——信用成本已在贡献度分解中被扣除。

## E — 可执行步骤 (Execution)

### 1. 计算目标银行的RORWA

- 从年报获取"归属于母公司股东的净利润"（或母公司净利润）
- 从年报"资本管理"或"风险管理"章节（通常在报表附注中）获取"风险加权资产"总额
- 若年报未直接披露风险加权资产，可从"各级资本充足率"反推：风险加权资产=核心一级资本净额÷核心一级资本充足率
- 计算：RORWA = 净利润 ÷ 风险加权资产 × 100%
- 完成标准：得到目标银行最近3年的RORWA（至少最近1年），并与9家银行历史基准对比（招行2.4%为标杆，平安~1.0%为警戒线）
- 判停点：若银行未上市/数据完全不可得，回退到以ROA和不良指标做替代判断，但需标注"缺乏RORWA数据，结论置信度打折"

### 2. 计算内生性增长底线R_min

- 获取参数：H=核心一级资本充足率（年报中找"核心一级资本充足率"），Ta=预估风险资产增速（可用近3年复合增速或当年监管口径下的行业均值10%左右），p=当年分红率（现金分红÷净利润）
- 代入公式：R_min = (H × Ta) / (1 - p)
- 示例：H=11.95%（招行2019），Ta=9%，p=33%→R_min=(0.1195×0.09)/0.67=1.59%
- 完成标准：得到R_min的精确数值，并判断实际RORWA是否≥R_min
- 判停条件：如果实际RORWA远高于底线（>1.5倍），可直接判定内生性增长无虞；如果接近或低于底线，需进一步做贡献度分解

### 3. 做RORWA贡献度三分法分解

- 分解为三项贡献度（以风险加权资产为分母）：
  - 扣除信用成本后净利息收入贡献度 = (净利息收入 - 信用减值损失) ÷ 风险加权资产
  - 手续费和佣金收入贡献度 = 手续费及佣金收入净额 ÷ 风险加权资产
  - 费用支出贡献度 = 业务及管理费 ÷ 风险加权资产（取正值为费用消耗）
- 完成标准：能用2-3句话解释"这家银行的RORWA是高是低，主要赢在哪一项，输在哪一项"
- 关键判据：扣除信用成本后净利息收入贡献度是区分银行的第一性指标——招行2.90% vs 民生0.69%，差距来自负债成本而非息差数值

### 4. 做纵向趋势判断（至少3年）

- 计算目标银行连续3-4年的RORWA趋势：上升/持平/下降
- 同期对比横向：拿出1-2家对标银行同期的RORWA趋势做参照（如目标银行vs招行、vs行业均值）
- 完成标准：能回答"RORWA的趋势和拐点在哪里？是持续改善还是恶化？"
- 关键信号：RORWA拐点通常比ROA拐点早1年出现（书中招行RORWA在2016年见底回升，ROA在2017年才确认拐点）

### 5. 综合结论：内生性增长判定

- 综合步骤2-4的结果，给出三档判定：
  - **绿色（内生性增长）**：实际RORWA > R_min 且连续3年趋势不下降 → 无需融资，可长期持有，分红可持续
  - **黄色（临界依存）**：实际RORWA ≈ R_min 或趋势在下降 → 需降低分红率或放慢资产扩张速度来维持核充率，警惕融资公告
  - **红色（价值毁灭）**：实际RORWA < R_min 且逐年下降 + 历史上频繁股权融资 → 参见平安银行模式，应回避
- 完成标准：给出明确的投资建议分级，并附RORWA和底线公式的数值支撑

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 分析非银行金融机构（保险、券商、信托）——RORWA和资本充足率是银行特有的监管体系
- 用户只想快速比PE/PB/股息率，不关心内生性增长→应先询问是否需要深入基本面分析，而非直接套用
- 数据严重残缺（连风险加权资产都无法推算）时强行建模型→应标记不确定性而非填补数据
- 仅比较两家银行的市值或净利润总量，而不涉及"效率"维度→规模对比不是RORWA的适用场景
- 用户问的是"某个宏观政策对银行板块整体影响"而非挑具体银行→RORWA是个股筛选工具，不适合板块层面的大颗粒度分析

### 作者在书中警告的失败模式

- **平安银行式的价值毁灭（ce09）**：只看净息差排名容易被高息差误导，忽略背后高信用成本的吞噬。高息差→给高风险客户放高利率贷款→不良率高→信用减值损失大→扣除后净利息收入贡献度极低→RORWA垫底→核充率下降→频繁再融资→摊薄股东权益→每股收益停滞不前。平安银行累积4年分红抵不过一次融资，是这一模式的典型案例。投资者最常见的误判：看见高息差=好银行，但赚到的利润全部被不良和再融资稀释。

- **"净利润增长=好银行"的陷阱（ce05）**：只看净利润增速会被信用减值调节蒙蔽。银行在营收丰年多提减值、在营收歉年少提减值来平滑利润。必须看"拨备前利润增速"（剔除信用减值调节），并用RORWA做效率验证——如果净利润增长但RORWA持续下降，说明增长是靠规模堆出来的，不是靠效率提升。

- **忽略核充率的恶化信号**：核心一级资本充足率是一个非常容易被忽略的重要指标——"表面上看这个指标不直接和净利润增速挂钩，每年变化也不大"。但作者将其比作"汽车油表的警告线"——一旦低于监管线（系统重要性银行8.5%），银行只能降速或再融资。连续3年核充率小幅下降0.1-0.2%的银行，趋势比绝对值更危险。

### 作者的盲点 / 时代局限

- **数据截止2019年，未覆盖疫情后银行分化加剧**：本书出版于2021年，核心RORWA数据用2016-2019年。2020年疫情后LPR大幅下调→净息差全行业压缩→RORWA普遍下降，且银行业分化进一步加剧。使用本skill时应代入最新的2022-2024年年报数据重新计算RORWA和底线，而非套用书中的横比结论。
- **作者是招商银行的长期重仓持有者**：全书以招行为标杆（自述持仓超10年、6年10倍），在RORWA分析中将招行设为"最优解"。这不必然错误（招行确实在RORWA上领先），但分析其他银行时应有独立验证——某些银行虽然RORWA低于招行，但如果满足自己的底线条件（如兴业2019年RORWA 1.29% > 底线1.25%），仍然是可投资的内生性增长银行。
- **公式简化假设的局限**：底线公式假设核心一级资本完全由未分配利润补充，实际上核充率中还包含"其他"项（如其他综合收益、外币折算差额等）。作者自己指出建行2019年因"其他"项大幅增加，在RORWA不够的条件下仍然实现了核充率上升。因此公式计算结果应视为"必要条件（不满足必融资）"而非"充分条件（满足一定不融资）"。
- **行业层面的宏观假设**：公式中Ta（风险资产增速）的选择带有主观判断。作者在书中用了10%作为标准假设（与名义GDP增速匹配）。如果中国名义GDP增速降到5%，银行风险资产增速可能同步下降到6%-7%，则RORWA底线也会降低——当前的底线判断依赖于对未来经济的合理预测，不是固定数字。

### 容易混淆的邻近方法论

- **混淆RORWA与ROA**：ROA=净利润÷总资产，RORWA=净利润÷风险加权资产。表面区别只是分母不同，但深层区别巨大——同样7万亿元总资产，招行的风险加权资产只有3.86万亿元（占比55%），兴业超过5万亿元（占比>70%）。ROA相同的两家银行，RORWA可能差一倍。RORWA捕捉的是"监管约束下的真实盈利能力"，ROA捕捉不到这一层。
- **将净息差排名等同于RORWA排名**：净息差一样不等于RORWA一样。平安银行净息差曾名列前茅，但扣除信用成本后的净利息收入贡献度是行业最低梯队。RORWA=P&L完整贡献度之和，包含手续费和费用两项，净息差只是利息净收入维度的一个侧面。
- **忽略高级法与权重法的RORWA不可比**：工、农、中、建、交、招行6家获批使用信用风险内部评级高级法，其他银行只能用标准化的权重法。高级法下，零售占比高、资产质量好的银行可以显著降低风险加权资产→RORWA被抬高。因此招行的RORWA对平安银行/民生银行的"2倍"优势中，有一小部分来自计量方法差异，不完全来自经营效率差异。跨方法比较时应将此因素纳入考量。
