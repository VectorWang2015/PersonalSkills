# 净息差拆解框架

## R — 原文 (Reading)

> 净息差=净利息收入÷总生息资产。相同净息差的银行，有的可能是依靠给高风险客户放贷获取高息差，有的可能是依靠低获取成本负债给低风险客户放贷达成的高息差，显然这两种银行获取相同的息差所冒的风险是完全不同的。对于顶级投资者，除了能够分辨净息差的含金量外，还需要对银行净息差的趋势进行预测。
>
> — 价投谷子地，《看透银行》第4章

> 扣除信用成本后净利息收入最终反映的实际是银行的负债成本控制能力。同样是净息差2.5%，招行可能资产平均成本只有4.4%，负债成本1.9%；而平安银行可能是资产平均成本5.4%，负债成本2.9%。金融行业的常识就是高利率对应高风险。
>
> — 价投谷子地，《看透银行》第8章

---

## I — 方法论骨架 (Interpretation)

净息差拆解框架的核心是：**净息差的数值不能孤立看，必须从资产端和负债端双向拆解，并且扣除信用成本后才能判断真正的"含金量"**。

具体分为三层分析：

**第一层：净息差总量**。计算净利息收入÷总生息资产，得到净息差的绝对值。这一步只是起点，不能作为结论。

**第二层：来源拆解（资产端+负债端）**。将净息差拆为"生息资产平均收益率"和"计息负债平均成本率"两段。资产端看贷款收益率（零售vs对公）、投资收益率、同业资产收益率的组成结构；负债端看存款成本（活期vs定期）、同业负债成本、应付债券成本的占比和利率水平。两段同时拆解才能看出净息差到底"赚在哪"。

**第三层：扣除信用成本**。净息差高的银行，如果其高息差来自给高风险客户放高利率贷款，则信用减值损失也会更高。计算"扣除信用成本后净利息收入÷生息资产"，以此判断息差的真实含金量。负债成本低→低风险定价→低信用成本的银行，才是真正高质量息差的拥有者。

关键区分：**低负债成本驱动的高息差**（可持续，有护城河）vs **高风险贷款驱动的高息差**（经济下行时信用成本飙升，息差被吞噬）。两类银行的净息差数值完全可能相同，但内在风险截然不同。

## A1 — 书中的应用 (Past Application)

### 案例 1: 招商银行 vs 交通银行 净息差全面拆解

- **问题**: 为什么招行的净息差比交行高出约70个基点？差距从哪里来？
- **方法论的使用**: 作者选取2017年年报数据，从资产端和负债端同时拆解。资产端：招行生息资产平均收益率4.06%，交行3.83%→差距来自招行零售贷款占比近50%（交行仅30%左右），且招行按揭仅占零售贷款的46.7%，信用卡+消费贷占50%以上（交行按揭占63.6%）。负债端：招行负债综合成本1.77%，交行2.39%→差距核心在招行存款成本仅1.27%，交行1.86%，根源是招行活期占比远高于交行。
- **结论**: 招行净息差领先来源于双重优势——资产端更高收益的零售贷款（且风险更低），负债端更低的存款成本（活期占比高）。金融业的常识是高利率对应高风险，但招行的零售贷款恰恰做成了"高收益低风险"。
- **结果**: 招行净息差2.43%，交行约1.73%，差距约70bp。这一优势在2016-2018年的4家银行对比中持续保持，且招行是9家银行中唯一RORWA连续4年上升的银行。

### 案例 2: 9家银行"扣除信用成本后净利息收入贡献度"横向对比

- **问题**: 不同银行的净息差看起来差距不大（四大行和股份行都约2%-2.5%），但为什么实际利润差异巨大？
- **方法论的使用**: 作者计算2019年9家银行（工农中建交+招浦民兴平）的RORWA贡献度分解，单独拉出"扣除信用成本后净利息收入贡献度"这一指标（=净利息收入减去信用减值损失后的余额÷风险加权资产）。
- **结论**: 差距触目惊心——招行高达2.90%，而民生银行仅0.69%，不到招行零头。四大行和招行此指标显著高于其他股份行，原因是负债成本低→无需以高风险贷款冲息差→信用成本低→扣除后的净利息收入贡献度领先。平安银行净息差虽名列前茅，但信用成本高达2.39%-3.08%，扣除后信贷业务基本赔钱。
- **结果**: 净息差"含金量"才是区分银行优劣的关键。负债成本控制能力是银行竞争力的一性原理——谁能用更便宜的钱放给更安全的客户，谁就是真正的赢家。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. **横向对比两家银行**：用户问"凭什么A银行的净息差比B银行高这么多？这能持续吗？"→需要拆解资产端和负债端，找出差距来源，判断是否可持续。
2. **单家银行的盈利能力深度分析**：用户问"这家银行的挣钱能力到底怎么样？"→不能只看净息差数值，要拆它的生息资产结构和计息负债结构，识别是"高质量"还是"高风险"息差。
3. **宏观利率环境变化下的银行筛选**：用户问"现在LPR持续下调，哪家银行能扛得住？"→活期占比高、负债成本稳定、资产端定价有护城河的银行，在降息周期中韧性更强。
4. **避开"看起来挣钱实际上不挣钱"的陷阱**：用户被高息差数据吸引（如某银行净息差同业排名第一）→需要计算扣除信用成本后的真实净利息收入，识破"平安银行式"的价值毁灭模式。
5. **银行股投资组合构建时的盈利能力排序**：用户问"在银行板块里，给盈利能力排个序"→RORWA的"扣除信用成本后净利息收入贡献度"指标是比净息差更优的排序工具。

### 语言信号

- "净息差怎么拆解"
- "为什么XX银行挣钱比YY银行多"
- "净息差的含金量怎么判断"
- "高息差就一定是好银行吗"
- "扣除信用成本后的净利息收入"

### 与相邻 skill 的区分

- 与 `liability-cost-analysis` 的区别：负债成本评估框架只拆负债端（存款成本、同业成本），净息差拆解框架同时拆资产端和负债端，并引入信用成本维度做最终含金量判断。前者是"成本视角"，后者是"盈利全景视角"。
- 与 `rorwa-evaluation` 的区别：RORWA评估框架用净利润端的完整拆解（净利息收入+非息收入-费用-信用成本-税），净息差拆解框架聚焦于"生息资产的利息净收入"这一条线，是RORWA分解中的"扣除信用成本后净利息收入贡献度"子模块。
- 与 `rate-change-impact` 的区别：加息降息影响分析聚焦利率环境变化对息差的边际影响，净息差拆解框架是静态拆解（给定一个时点数据，息差从哪来）。两者互补：先用本skill拆解现状，再用利率影响skill判断变化方向。

## E — 可执行步骤 (Execution)

1. **获取数据，分层拆解净息差**
   - 从年报"净利息收入"小节获取生息资产平均余额、利息收入/支出、各类资产负债的平均收益率/成本率
   - 计算净息差 = 净利息收入 ÷ 总生息资产平均余额
   - 拆为资产端：各类生息资产（贷款/投资/同业/存放央行）的占比和收益率
   - 拆为负债端：各类计息负债（存款/同业/应付债券）的占比和成本率
   - 完成标准：能说出"净息差X%，其中资产端贡献Y%，负债端成本优势贡献Z%"

2. **计算扣除信用成本后的净利息收入贡献度**
   - 取得信用减值损失金额（利润表中的"信用减值损失"）
   - 计算：扣除信用成本后净利息收入 = 净利息收入 - 信用减值损失
   - 计算贡献度 = 扣除信用成本后净利息收入 ÷ 总生息资产（或风险加权资产）
   - 完成标准：能回答"表面净息差和扣除信用成本后的净息差差多少？"
   - 判停条件：若信用成本占比极低（<0.3%），说明该银行资产质量风险不大，可简化分析

3. **定性判断净息差的可持续性和含金量**
   - 判断净息差来源：负债成本驱动（低成本活期占比高）→可持续；资产端高风险定价→需警惕信用成本反转
   - 纵向看趋势：净息差过去3年是在改善还是恶化？资产/负债端哪一端在驱动变化？
    - 完成标准：能定性回答"这个净息差的质量如何？未来是扩大还是收窄？"并给出两端各自的趋势判断

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 仅比较两家银行的资产规模或市值大小，而不涉及盈利效率——这是规模对比，非净息差拆解
- 用户只问银行的净利润增速或ROE，没有问到"为什么能挣到这样的息差"——应先查净利润质量分析类skill，非本skill的触发条件
- 分析非银行金融机构（保险、券商）的盈利模型——它们的业务结构不同，不存在"净息差"概念
- 数据不充分（缺乏负债端分项数据）时强行给结论——应要求用户补充数据而非猜测

### 作者在书中警告的失败模式

- **"躺着吃息差"论（ce11）**：大众用"贷款利率6% - 存款利率1.5% = 4%息差"这种粗暴减法得出结论，完全忽略了银行负债结构中还有成本远高于存款的同业负债和应付债券。实际平均净息差仅约2%，甚至低于美国富国银行的2.9%。一旦分析者犯下"过度简化"的认知错误，就会严重高估所有银行的盈利能力，无法区分优劣。
- **对称加减息误判（ce02）**：投资者想当然认为各档期存贷款利率同步升降时息差不变。实际上活期存款利率通常不跟随基准利率调整，导致对称加息时活期占比高的银行净息差仍扩张，降息时息差收缩。忽略负债结构的非对称性，是净息差趋势预测中最危险的认知陷阱。

### 作者的盲点 / 时代局限

- **数据截止2019年，未覆盖疫情后环境**：本书出版于2021年，核心年报数据截止2018-2019年。2020年后的持续LPR下调、房地产行业调整导致的资产荒、净息差全行业收窄等重大变量未覆盖。当前环境下几乎所有银行的净息差都在压缩，分析时应结合最新数据（至少2022-2024年年报数据），不能刻舟求剑地套用书中2017年的数字。
- **作者是招商银行的长期重仓持有者（自述持仓超10年、6年10倍）**：全书以招行为"最优解"标杆，净息差拆解中可能选择性强调招行优势维度（负债成本低），而对招行资产端定价相对保守（贷款利率并不激进）的约束讨论不足。分析其他银行时应独立判断，不被作者的"招行滤镜"带偏。
- **对风险定价能力的假设过于乐观**：作者认为零售贷款天然是"高收益低风险"，这一结论依赖中国至今没有个人破产法、央行征信系统完善的前提。若未来个人破产法出台、经济下行导致零售不良率上升（如2022年后信用卡不良率上行），零售贷款的"低风险"特征可能被打破。

### 容易混淆的邻近方法论

- **不区分信用成本的净息差排名**：直接将银行按净息差从高到低排序是危险的。平安银行净息差名列前茅，但扣除信用成本后所剩无几。做任何银行间比较时，必须先扣除信用成本再做净息差分析。
- **混淆净息差与净利差**：净利差 = 生息资产平均收益率 - 计息负债平均成本率；净息差 = 净利息收入 ÷ 总生息资产。两者通常净息差略高于净利差（因自有权益不计息但参与生息），但数值差异一般不超20bp。新手容易混用。
- **用制造业现金流分析框架套到银行**：银行的经营活动现金流为负不等于造血能力不足，因为银行原料（存款）、产品（贷款）和利润都是现金形态。净息差分析依赖的是资产负债表+损益表，不应引入经营现金流指标来"佐证"净息差结论。
