# 避雷三阶段法 — 财务风险防范体系

## R — Reading (原文)

> "不踩雷的最好方法其实是'绕路走'，而不一定非要把雷排出来。只要发现异常数据且找不到逻辑解释，就可以排除该公司，不需要找到造假证据。这是一个冗余安全系统。"（第15章,前言/第12页）

> "资产负债表在三张报表中起到桥梁作用，任何财务问题都将在资产负债表上留下痕迹。数字越大，影响越大。大数字就是资产负债表中相对于整张报表明显更大的科目，需要重点关注。"（第15章,第282页）

> "只要不是确定好的公司，我们就应将其排除在投资池外。只要一家公司超出了你的能力圈，就意味着公司中有你看不到或无法理解的风险。"（第15章,第288页）

## I — Interpretation (自述)

避雷三阶段法不是教你成为"排雷专家"(找出造假的证据)，而是建立一个**冗余安全系统**——在异常值出现时就绕道走，不需要等到真相大白。

**核心哲学：绕路而非排雷。**
- 不需要找到造假证据。只要发现"数据异常+逻辑说不通"的组合，就可以排除。
- 这是冗余设计——可能因此错过少数"数据异常但其实是好公司"的标的(如恒瑞医药净现比下降但股价上涨)，但保护你在绝大多数雷区全身而退。
- 类比：不赚这个钱vs踩一次雷。纪律可能让你错过牛股，但侥幸心理能让你血本无归。

**三阶段递进逻辑**：
- **事前**：七步成诗法的"魔鬼版"——不是找好公司，而是逐筛挑刺。每一筛就是一个判定门，不通过马上亮红灯。
- **事中**：如果不死心还要继续，打开资产负债表直接看结构。"大数字"是快速定位风险的导航系统。
- **事后**：诚实面对自己——这家公司你真的看懂了吗？如果答案是"不确定"或"看不懂"，直接排除。

**事前七道筛子(魔鬼版七步成诗法)**：
1. **扣非ROE过滤**：ROE是否被非经常性损益粉饰？（扣非vs不扣非ROE差距>50%→红灯）
2. **现金收入比率过滤**：营业收入有现金支撑吗？（<0.6→红灯）
3. **毛利率逻辑过滤**：毛利率显著高于同行但找不到合理商业逻辑→红灯(产品同质化却高毛利=造假最可能突破口)
4. **净利润三维过滤**：净现比<0.5超过两年？扣非占比剧烈波动？→红灯
5. **周转率恶化过滤**：存货/应收周转率多年持续下降+绝对额上升→红灯
6. **有息负债率+偿债安全过滤**：有息负债率>30%且现金/带息债务<1→红灯
7. **存贷双高过滤**：货币资金+有息债务双高且无合理解释→红灯

**事中阶段：看资产负债表"大数字"**
- 不需要看全部科目，只看占比突出的"大数字"
- 流动资产大数字：货币资金/应收款项/存货——各占总资产比重多少？结构健康吗？
- 非流动资产大数字：固定资产/在建工程/无形资产/商誉——无形资产/商誉占比过高是危险信号(价值不确定性大)
- 负债大数字：短期借款/应付债券——是否存在存贷双高？
- 黄金原则：用一家公认好公司(如海天味业)的资产负债表结构做对标，任何严重偏离都需要合理解释。

**事后阶段：诚实三分类**
- **确定好**：少之又少，大部分只是"看起来不错"
- **确定不好**：七筛红灯多+大数字异常+逻辑说不通
- **看不懂/拿不定主意**：这是大多数情况！→**直接排除**，不要在不确定的事情上赌

## A1 — Past Application (书中案例)

**A公司(乐视)完整避雷演示**：
- 事前：扣非ROE>15%通过第一筛→现金收入比率多年<60%(第二筛红灯)→毛利率从40%+降至14%("增收不增利"，第三筛红灯)→净利率<2%+净现比剧烈波动(第四筛红灯)→有息负债率>36%+现金/带息债务<1(第六筛红灯)。
- 事中：流动资产大数字=42.69亿应收款(占总资产25%！多属下游欠款，易减值)。非流动资产大数字=48.8亿无形资产(影视版权为主，价值难评估)。流动负债大数字=17.35亿短期借款→扣除后净现金仅10亿，对应收42亿完全不够。其他非流动负债34.71亿主要为老板夫妇借款→上市公司向自然人借大钱="极度缺钱"信号。
- 事后：A公司确定不好吗？退回到2015年大多数人选择"看不懂"→只要不是"确定好"，一律排除。事后证实爆雷。

**B公司(康美药业)专注异常点**：
- 事前：毛利率比同行高近一倍但中药饮片无差异化→逻辑说不通(红旗)。净现比连续4年<50%(2015年仅18.46%)→致命红旗。
- 事中：货币资金341亿+有息负债197亿="存贷双高"(融资成本5-6%)→有300多亿却借高息钱→逻辑说不通。存货周转率10年持续下降→产品销路有问题。
- 事后：无法确定"确定不好"，但绝不是"确定好"→排除。事后证实虚增货币资金300亿+营收88亿。

**海天味业对标(健康资产负债表结构)**：流动资产=45亿现金+10亿存货(正常)+16亿其他流动资产(银行理财，可接受)。非流动资产=32.64亿固定资产(厂房设备，正常)。负债端=全为无息负债(11亿预收款+5.85亿应付款)，无有息借款。完美对标的健康结构。

## A2 — Future Trigger ★

**场景1**: 用户分析过程中发现异常数据(如毛利率高得离谱、净现比突然暴跌)不知如何处理→启动避雷三阶段法判断是否该排除。
**场景2**: 用户问"怎么判断一家公司是否造假""这个数据是不是有问题"→七筛逐层过，任何一项红灯触发"逻辑说不通"判断。
**场景3**: 用户看到"存贷双高"现象→这是最典型红旗，需立即警示用户并展开详细分析。
**场景4**: 用户投资了一只股票但心里不踏实"总觉得哪里不对"→用避雷法系统性地排查一遍。

**语言信号**: "避雷" / "排雷" / "造假" / "异常数据" / "不对劲" / "存贷双高" / "这个数据是不是有问题" / "怎么判断财务造假" / "大数字"。

**与相邻skill区分**: 本skill专注于"找问题"和"排除"，是七步成诗法的反面对照。`zong-he-pan-duan`是正面筛选(找好公司)，本技能是反面排除(踢坏公司)。两者应配合使用：先用避雷法排雷，再用综合判断法选优。

## E — Execution (可执行步骤)

**事前阶段 Step 1-7 — 七筛过滤**：
依次检查：
1. 扣非ROE vs 不扣非ROE差距 → >50%红灯
2. 现金收入比率(最近5年) → <0.6任一或多年<0.8 → 红灯
3. 毛利率 vs 同行+商业逻辑合理性 → 高出行50%无逻辑解释 → 红灯
4. 净利润现金含量(净现比) → <0.5超过2年 → 红灯
5. 关键周转率趋势(存货/应收) → 连续3年+下降且绝对额上升 → 红灯
6. 有息负债率 → >30%且现金/带息债务<1 → 红灯
7. 存贷双高检查 → 货币资金+有息负债均庞大且无合理解释 → 红灯

判停：任一红灯触发"逻辑说不通"，可直接排除。若全部通过→进入事中。

**事中阶段 Step 8-10 — 资产负债表大数字扫描**：
8. 扫描资产端：列出占比>15%的科目，判断每项的减值风险和真实性
9. 扫描负债端：区分有息/无息，关注是否有"其他"类大数字(其他非流动负债占比大→查附注)
10. 结构对标：找一家同行业公认好公司的资产负债表对比结构

判停：若资产结构严重偏离健康对标且无合理解释→排除。

**事后阶段 Step 11 — 诚实三分类**：
11. 问自己：这家公司是(1)确定好 / (2)确定不好 / (3)看不懂？
判停：答案不是(1)→直接排除。大部分情况答案是(3)，不要为难自己。

**Step 12 — 输出**：给出避雷结论(通过/排除/+理由)，标注所有红旗和逻辑矛盾点。

## B — Boundary (边界)

**不适用场景**:
- 对已确认为"好公司"的标的做深度投资分析→避雷法只做排除不做推荐
- 已确认造假后的维权咨询→不涉及法律层面
- 替代正面财务分析→应与七步成诗法配合使用(先排雷再分析)
- 要求给出"排雷证据"(举证造假)→本方法只找"异常值+逻辑不通"，不做证据收集

**需要注意的过度排除风险**:
- 恒瑞医药(创新药龙头)2016年后净现比<50%被七筛过滤，但股价同期涨了几倍——好的公司也可能被避雷法误伤。这是纪律的代价。
- 某些行业天然具有低现金含量特征(工程承包的分阶段付款、地产的项目制结算)，不能一刀切套用0.8阈值。
- "逻辑说不通"的主观性——你认为说不通也许只是因为你不懂这个行业。能力圈越窄，被排除的公司越多。

**作者的时间局限性**: 案例写于2021年末，无法预测2022年之后地产行业全线爆雷和教培政策的颠覆。在行业系统性崩塌面前，再好的避雷法也无法预测政策黑天鹅。
