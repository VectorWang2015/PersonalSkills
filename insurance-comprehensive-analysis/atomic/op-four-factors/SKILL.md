# 营运利润四因子拆解

## R — 原文 (Reading)

> 营运利润带着在可接受度和真实度间寻找平衡的使命"呱呱坠地"，要在内含价值评估标准和一般会计准则间极力斡旋。以财务报表净利润为基础，剔除短期波动性较大的损益表项目和管理层认为不属于日常营运收支的一次性重大项目。
>
> — 东先生，《读懂保险股》第6章

> 在这套讨好内含价值评估标准的表述中，营运利润是按来源进行分拆的，由剩余边际摊销、净资产投资收益、息差收入和营运偏差构成。剩余边际摊销对应销售，净资产投资收益和息差收入对应投资，营运偏差对应营运。
>
> — 东先生，《读懂保险股》第6章

> 新业务增长放缓，最终将拖累营运利润的增长，随后拖低公司的分红率，从而影响投资者拿到手的真金白银，最后传导到对公司的估值。
>
> — 东先生，《读懂保险股》第6章

## I — 方法论骨架 (Interpretation)

营运利润体系是保险公司财务分析的核心工具——它在一般会计准则（净利润）和内含价值评估标准之间架起一座桥梁，让投资者能用一种"接近内含价值逻辑但又不那么精算密集"的方式理解保险公司的盈利能力。

营运利润拆解为四个因子，分别映射保险公司商业模式的三个环节：

**四因子的定义与对应关系：**

1. **剩余边际摊销** → 对应**销售**。剩余边际是保单签发时"首日利得"被强制计入准备金的暂存利润，逐年摊销释放为利润。它是营运利润中的"承保利润"部分，驱动因子是存量保单的剩余边际余额及其摊销模式（保障类摊销快、储蓄类摊销慢）。

2. **净资产投资收益** → 对应**投资**（自有资本部分）。基于净资产按 5% 投资收益率假设计算的收益。这部分对应的是保险公司自有资本的投资回报，5% 的假设与内含价值保持一致。

3. **息差收入** → 对应**投资**（负债资金部分）。以负债对应的资产按 5% 投资收益后，扣除准备金要求回报（"负债成本"）后多赚的部分。本质上就是保险公司的"利差"——投资收益率超过预定/准备金利率的差额。准备金要求回报约 3.5%，当投资收益率超过 2% 时开始赚取剩余边际，超过 3.5% 时开始赚取息差收入，超过 5% 后超出的部分不计入营运利润（仍计入净利润）。

4. **营运偏差** → 对应**营运**。实际运营支出（业务管理费、佣金、理赔等）优于精算假设的差额。正的营运偏差意味着公司运营效率高于预期，负的则意味着费用超支或退保率恶化。

**关键洞察：两种"到达"营运利润的路径。**

- 路径一（讨好一般会计准则）：从净利润出发，剔除短期投资波动（>5%的部分）、折现率变动、一次性重大项目 → 得到营运利润。
- 路径二（讨好内含价值）：从剩余边际出发，叠加投资侧两因子和营运偏差 → 得到营运利润。

两条路径殊途同归，构成"三套准则的钩稽关系"中最实用的一环。同时，剩余边际的进出水口结构（新业务价值贡献为进水、剩余边际摊销为出水）完美映射了内含价值中有效业务价值的结构，因此**剩余边际是有效业务价值在营运利润体系的"投影"**。

## A1 — 书中的应用 (Past Application)

### 案例 1: 中国平安 2019 年营运利润四因子拆解

- **问题**: 中国平安 2019 年税前营运利润 1005 亿，同比仅增 2.6%，增长几乎停滞，原因是什么？
- **方法论的使用**: 作者按四因子逐项拆解平安 2019 年 vs 2018 年的营运利润变化。剩余边际摊销 744 亿（+20%），仍是增长引擎；净资产投资收益 117 亿（+31%），受益于净资产扩大；但息差收入从 50 亿降至 39 亿（-22%），因公司主动降低息差水平以保持产品竞争力，影响延续到 2019 年；营运偏差从 217 亿骤降至 104 亿（-52%），因为持续加大科技、队伍建设等战略投入，以及保单继续率短期波动。
- **结论**: 平安营运利润增速骤降的根源不在剩余边际摊销（销售端依然健康，新业务价值贡献是摊销的 2 倍），而在于营运偏差的大幅收窄和息差收入的被动压缩。前者是主动战略投入（可控），后者是利率环境压制（不可控）。
- **结果**: 四因子拆解揭示了一家表面"增长停滞"的公司内部并非全面恶化——其销售发动机（剩余边际摊销+20%）依然强劲，问题集中在投资端和费用端，投资判断应区分"结构性恶化"与"周期性压力"。

### 案例 2: 新业务价值放缓→营运利润→分红→估值的传导链分析

- **问题**: 如果一家保险公司的新业务价值增速放缓，对投资者的真金白银有什么影响？
- **方法论的使用**: 作者以平安 2019 年数据追踪传导链：新业务价值扣偿后仅增 5%，考虑风险分散效应后甚至负增长（-3.53%），导致剩余边际摊销增长率从 2018 年的 25% 下降至 20%，摊销占总剩余边际余额的比例从 10.1% 降至 9.5%。
- **结论**: 营运利润中的"发动机"是剩余边际摊销，而剩余边际摊销的增长率取决于新业务价值这一"进水口"。进水口减速→出水口增速下降→营运利润增速下降→分红能力减弱→估值承压。那些声称"即使不再产生新业务，内含价值也有 x% 增长"的论调是价值陷阱。
- **结果**: 营运利润四因子不仅是当期利润的拆解工具，更是未来盈利趋势的前瞻指标——剩余边际摊销的同比增速变化，比当期营运利润绝对额更值得关注。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. **分析保险公司利润质量**：用户看到净利润大幅波动（如投资收益暴涨暴跌），想了解剔除短期波动后的真实经营趋势 → 需要通过净利润→营运利润的调整路径，看清四因子的贡献结构。
2. **横向对比两家险企的盈利模式**：用户问"为什么平安和大保的利润结构不一样" → 需要拆解两家公司的四因子占比，判断谁是"销售驱动型"（剩余边际摊销占比高）、谁是"费控驱动型"（营运偏差占比高）。
3. **判断保险公司利润可持续性**：用户看到净利润高增长但不确定能否持续 → 通过四因子拆解看增长来自哪里：剩余边际摊销增长（可持续，取决于新业务价值）vs 息差收入增长（不可持续，取决于利差环境）vs 营运偏差改善（中性，需看是一次性还是结构性）。
4. **评估利率环境变化对保险公司的影响**：用户关心降息周期下保险股的盈利韧性 → 通过息差收入因子量化利率下行对营运利润的冲击（准备金要求回报固定时，投资收益率每降 1% 对息差收入的影响）。
5. **保险股估值前的盈利归一化**：用户在做 DDM 或 PEV 估值前需要"正常化"的盈利数字 → 营运利润比净利润更适合作为估值基础（剔除了不可持续的投资波动和一次性项目）。

### 与相邻 skill 的区分

- 与 `ev-decomposition`（内含价值拆解）的区别：内含价值拆解从 DCF 框架看保险公司"值多少钱"（EV = ANA + VIF - CoC），营运利润四因子拆解从损益表框架看保险公司"挣了多少钱"。二者互补：EV 是存货思维，OP 是流量思维。剩余边际是 VIF 在营运利润体系的投影。
- 与 `residual-margin-analysis`（剩余边际分析）的区别：剩余边际分析聚焦于准备金的蓄水池逻辑（余额、摊销、新业务价值贡献的进出水口），营运利润四因子在此基础上叠加了投资侧（净资产投资收益+息差收入）和费控侧（营运偏差），形成完整的利润来源视图。
- 与 `three-standards-framework`（三套准则关系理解）的区别：三套准则框架讲的是 GAAP vs EV vs OP 的宏观关系和哲学意义，营运利润四因子是其中的"OP"维度的具体执行方法论。前者是地图，后者是导航。

## E — 可执行步骤 (Execution)

1. **从净利润出发，调整为营运利润**
   - 获取净利润（损益表）
   - 加回/剔除三项：①短期投资波动（实际投资收益率与 5% 假设的差额中归属股东的部分）；②折现率变动对准备金的影响；③管理层认定的一次性重大项目（如税改清算损益）
   - 完成标准：得到集团层面和人身险业务层面的营运利润数字，与公司披露的营运利润核对一致
   - 判停条件：若被分析公司不披露营运利润调整明细（如中国人寿、新华保险），则跳过此步，直接基于披露口径或自行估算

2. **按四因子拆解营运利润来源**
   - 从财报"内含价值与营运利润分析"章节（或管理层讨论）获取四因子数据：剩余边际摊销、净资产投资收益、息差收入、营运偏差
   - 计算各因子占税前营运利润的比例，判断利润主导因子
   - 典型参考区间：优质人身险公司剩余边际摊销占比 65%-75%，净资产投资收益 10%-15%，息差收入 3%-5%，营运偏差 5%-15%
   - 完成标准：能说出"营运利润 X 亿，其中销售贡献 Y%、投资贡献 Z%、营运偏差贡献 W%"

3. **判断各因子的趋势与可持续性**
   - 剩余边际摊销：计算同比增速，与当年新业务价值增速对比（进水口 vs 出水口）→ 若新业务价值增速持续低于摊销增速，警惕蓄水池缩水
   - 息差收入：结合 10 年期国债收益率趋势判断→降息周期中息差收入大概率收窄
   - 营运偏差：区分结构性改善（如数字化降费）与一次性波动（如退保率异常）→ 查阅公司对偏差变动的解释
   - 完成标准：能回答"未来 2-3 年营运利润的增速中枢大概在什么区间？主要增长引擎是什么？最大风险因子是什么？"

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 仅询问内含价值或 PEV 估值倍数——营运利润是损益视角，不是资产负债表视角
- 分析财产险公司（如人保财险、平安产险）——财险保单期限短（1 年），不存在剩余边际摊销的概念，营运利润四因子框架仅适用于寿险/健康险等长期人身险业务
- 用户只想要保险公司的保费收入排名或市场份额——这是规模指标，非盈利质量分析
- 数据不充分（未披露营运利润四因子明细）时强行推算——只有中国平安和中国太保完整披露，其他公司需向用户说明数据限制

### 作者在书中警告的失败模式

- **价值陷阱**（ce01）：听到"即使保险公司不再产生新业务，内含价值也有 x% 增长"就盲目相信。实际上，新业务价值进水口一旦枯竭，剩余边际摊销终将衰减，最终传导至分红和估值。蓄水再大，无水也干。
- **5% 假设的"魔术"**（ce02）：营运利润体系的投资收益率假设统一为 5%，但实际投资环境未必能达成此收益。投资者不应将 5% 视为"无风险"的确定性收益，尤其当长端利率系统性下降时，5% 的假设面临压力测试。

### 作者的盲点 / 时代局限

- **数据截止 2019 年，未覆盖后疫情低利率环境**：本书以 5% 投资收益率假设为前提，但 2020 年后中国 10 年期国债收益率一度跌破 2.6%，保险公司的实际投资收益率距 5% 差距扩大，营运利润体系下的"息差收入"理论上可能归零甚至为负。当前分析必须结合最新利率环境。
- **仅平安和太保披露营运利润**：A 股四大险企中，中国人寿和新华保险截至出版时未披露营运利润。作者以平安为唯一范本的分析可能不具普适性，投资者自行估算其他公司营运利润时需保守假设。
- **IFRS17（2023 年生效）对营运利润体系的重塑未覆盖**：新准则下保险服务收入确认方式改变，剩余边际的概念被"合同服务边际(CSM)"取代，营运利润的调整口径可能变化，本书的分析框架需结合 IFRS17 下的新披露格式重新校准。

### 容易混淆的邻近方法论

- **将营运利润等同于"核心净利润"**：会计意义上的"扣非净利润"剔除的是非经常性损益，但保险公司的投资波动是经营的内生组成部分，不是"非经常性"。营运利润的调整逻辑更激进——它不仅剔除了一次性的，还剔除了经常性但不可预测的（投资收益>5%部分）。
- **混淆剩余边际摊销与"现金流"**：剩余边际摊销是会计利润概念，不代表当期现金流入。保险公司当期实际可用于分红的现金来源于调整净资产产生的真实现金流，而非剩余边际摊销这一"纸上利润"。
- **用杜邦分析拆解营运利润**：传统杜邦分析（ROE=净利率×周转率×杠杆）不适用于长期保单业务。保险公司的"利润"发生在不同时点（签发时确认剩余边际、逐年确认摊销），利润产生机制是跨期性的，不适合用资产周转率等当期指标分析。
