# 保险股估值方法选择 — 综合估值决策框架

## R — 原文 (Reading)

> 在估值上，老东认为……公司的价值，最合适的答案永远是：公司未来所有可以赚的自由现金流的折现值。因而，为了让我们的投资决策确定性更高，老东建议在行业和公司的思考上投入的时间的比重应该要大于估值，而不是颠倒过来。

> — 东先生，第5章 "内含价值评估标准"

> PEV本质上就是保险股的PB，只是把分母从会计净资产替换成了内含价值。PEV < 1意味着市场认为内含价值的假设过于乐观，PEV > 1则说明市场认可公司持续创造超额新业务价值的能力。

> — 东先生，第5章

> 有效业务价值实际上就是果树生命周期所有果实变现后盈余的折现，像一张可以预测利息的债券。这张债券的常规动作是每年释放11%的价值，远高于调整净资产投资所得的5%左右收益率。因此，有效业务价值在内含价值中的占比，可以反映每家公司内含价值"外强中干"的水平。

> — 东先生，第8章

> 保障型产品占比高的公司，剩余边际蓄水池更大、摊销释放更稳定，NBV率也更高——这是典型的"轻资产、高ROE"模式。反观以储蓄型产品为主的公司，利润大量锁定在保单终末期，有效业务价值虽然数字不小，但释放极慢，属于"注水的有效业务价值"。

> — 东先生，第8章 公司案例对比

> 就比如，如果一家保险公司把分红率定在60%，以调整净资产为分红基础，内含价值11%的增长中，仅有2%（约3.5%×60%）可以分给股东，剩余9%的增长将留在公司继续增值。但如果把60%的分红率换成以营运利润为分红基础，11%×营运利润/内含价值≈6-7%，股东实际到手的分红将大幅提升。

> — 东先生，第8章 "股息与分红政策"

## I — 方法论骨架 (Interpretation)

### 保险股估值的独特挑战

保险股不能简单地用PE估值（"费用前置利润后置"使PE系统性失真），也不能直接套用银行的PB估值（保险的净资产里藏着巨量的剩余边际未释放）。作者确立的核心估值框架是：

**PEV是主锚，但不迷信PEV。结合ROEV、NBV增长、股息率和保障/储蓄属性做综合判断。**

### 主锚：PEV — 保险股的"含金量标尺"

```
PEV = 股价 ÷ 每股内含价值
```

| PEV区间 | 含义 | 操作含义 |
|---------|------|----------|
| < 0.5 | 深度低估 | 需先排除"价值陷阱"（NBV负增长且无改善迹象），确认非价值毁灭后才考虑建仓 |
| 0.5 ~ 0.8 | 低估区间 | 具备安全边际，重点验证EV质量（VIF占比/释放率）和NBV增速是否触底回升 |
| 0.8 ~ 1.0 | 轻度低估 | 合理偏低，结合股息率判断——若股息率>4%且可持续，防御性配置价值凸显 |
| 1.0 ~ 1.5 | 合理区间 | 估值公允，需NBV增速>10%或ROEV>15%才具备超额收益吸引力 |
| > 1.5 | 溢价区间 | 市场给予高成长溢价，需NBV增速>20%+"保障型为主"才可维持，否则减仓 |

**PEV > 1 的溢价逻辑**：市场认为该公司不仅能守住存量内含价值的11%自然增长，还能持续创造超额NBV。PEV=1.5意味着市场认为未来NBV增速将远超行业平均——这是"成长股定价"，不是"价值股定价"。

### 辅助锚之一：ROEV — 内含价值的"内生回报率"

```
ROEV ≈ (期初调整净资产×3.5% + 期初有效业务价值×11% + 扣偿后NBV) ÷ 期初内含价值
```

- ROEV 5%~8%：债券型保险公司，缺乏成长性，应参考高股息策略（股息率>5%才具备吸引力）
- ROEV 10%~15%：正常成长型保险公司，PEV在1附近是合理估值
- ROEV 15%~20%：高速成长型，PEV 1~1.5倍可以接受
- ROEV > 20%：极度稀缺的"成长+质量"双优公司，PEV > 1.5也有合理性

### 辅助锚之二：股息率 — 保险股"落袋为安"的保底

保险公司的股息率是估值的"地板"。因为内含价值增长中约有3.5%来自调整净资产的真实投资收益，这部分通过分红可以实际到手。如果一家保险公司：
- 以**营运利润为分红基础**（如中国平安），股息增长更可持续
- 以**调整净资产为分红基础**（多数央企系），股息率天花板更低

```
保险股吸引力门槛（参考）：
  PEV < 1：股息率 > 2.5% 即可接受（增量来自EV增长带来的股价重估）
  PEV 1~1.5：股息率 > 3.5% 才能弥补估值下调风险
  PEV > 1.5：股息率 > 5% 才具备安全边际（因高PEV隐含下行风险）
```

### 辅助锚之三：NBV增长前景 — 决定PEV溢价/折价的方向

NBV是保险公司价值增长的"发动机"（详见 `atomic/nbv-analysis/SKILL.md`）。NBV增速的方向决定了PEV的变动方向：

| NBV增速 | PEV含义修正 | 估值操作 |
|---------|------------|---------|
| NBV增速>20% | PEV即使1.5也不算贵（高成长消化高估值） | 可接受适度溢价 |
| NBV增速5%~15% | PEV在0.8~1.2为合理区间 | 按估值中枢操作 |
| NBV增速负增长且未见底 | PEV即使0.5也不便宜（价值在持续毁灭） | 回避，等NBV增速转正再评估 |

**关键区分**：NBV增速下降但NBV率上升 → "减量增质"转型期，PEV低估可能是机会。NBV增速下降且NBV率也在下降 → 公司基本面全面恶化，低PEV是价值陷阱。

### 核心区分：保障型 vs 储蓄型保险公司的估值逻辑

这是第8章作者通过4家公司对比揭示的**保险股估值最重要的二元分叉**：

| 维度 | 保障型保险公司 | 储蓄型保险公司 |
|------|--------------|--------------|
| **典型代表** | 平安、太保（"大个险"战略） | 国寿、新华（分红/万能险为主） |
| **产品特征** | 长期健康险/寿险，死差+费差驱动 | 分红险/万能险，利差驱动 |
| **NBV率** | 35%~45%（高利润率） | 20%~32%（低利润率） |
| **EV质量** | VIF/EV占比高（>55%），VIF按11%释放快 | VIF/EV占比低（<45%），VIF"注水"释放慢 |
| **ROEV** | 15%~20%，增长质量高 | 8%~12%，增长依赖利差 |
| **剩余边际** | 蓄水池大、摊销快（保障型保单早期摊销多） | 蓄水池小、摊销慢（储蓄型均匀释放） |
| **利率敏感性** | 低（主要赚死差/费差，对利差依赖低） | 高（主要赚利差，利率下行直接压缩利润） |
| **合理PEV区间** | 0.8 ~ 1.5（可给予更高溢价） | 0.5 ~ 1.0（应给予折扣） |
| **股息率要求** | 2.5%+ 即可（成长性弥补） | 4%+ 才具吸引力（缺乏成长性） |

**投资含义**：
- 保障型公司在PEV<1时是"打折的成长股"，安全边际+成长性双重优势
- 储蓄型公司在PEV<0.5时才具备真正安全边际（因为EV中的VIF可能大面积"注水"）
- 当利率进入下降周期，储蓄型公司的PEV应打更大折扣（利差损风险上升）

### 综合估值决策矩阵

将PEV、ROEV、NBV增速、保障/储蓄属性、股息率五维合并，形成决策矩阵：

| PEV | ROEV | NBV增速 | 保障型 | 储蓄型 | 股息率 | 综合判断 |
|-----|------|---------|--------|--------|--------|---------|
| < 0.8 | > 15% | > 15% | ★★★ 重仓 | ★★ 标配 | 任意 | 强买入 |
| < 0.8 | 10%~15% | 5%~15% | ★★ 加仓 | ★ 观察仓 | > 3% | 买入 |
| < 0.8 | < 10% | < 0% | ★ 小仓试探 | 回避 | > 4% | 仅股息策略 |
| 0.8~1.2 | > 15% | > 15% | ★★ 标配 | ★ 防御仓 | > 3% | 持有/加仓 |
| 0.8~1.2 | 10%~15% | 5%~15% | ★ 持有 | 观望 | > 3.5% | 持有 |
| 0.8~1.2 | < 10% | < 5% | 观望 | 回避 | 任意 | 不参与 |
| > 1.5 | > 20% | > 20% | ★ 可持有 | ★ 减仓 | > 3% | 持有但不加 |
| > 1.5 | 10%~15% | < 10% | 减仓 | 清仓 | 任意 | 卖出 |

## A1 — 书中的应用 (Past Application)

### 案例 1：4家险企PEV横向对比 — 谁贵谁便宜？（第8章）

- **问题**：中国平安、中国人寿、中国太保、新华保险，4家公司按PEV排，谁最被低估？
- **方法论的使用**：作者将4家公司的内含价值与市值做PEV对比。关键步骤——统一贴现率口径后再比较。平安和太保使用11%贴现率，国寿使用10%，新华使用11.5%。国寿的PEV表面更低，但如果将国寿的贴现率从10%统一调至11%，内含价值因贴现率提升而缩水，PEV实际上升。
- **结论**：不能直接拿年报PEV排大小。保障型+高贴现率（保守）的公司（如平安11%），PEV即使看起来不低，也是被低估的（因为内含价值假设更保守）。储蓄型+低贴现率（激进）的公司（如国寿10%），PEV即使看起来低，也可能并不便宜。
- **结果**：PEV的比较必须分两步走——先统一假设口径，再区分保障/储蓄属性。

### 案例 2：有效业务价值占比揭示的估值差异（第8章/f20）

- **问题**：为什么平安PEV约1.3倍市场仍愿意给溢价，而国寿PEV < 1市场反而觉得贵？
- **方法论的使用**：作者拆解4家公司内含价值的内部结构——有效业务价值占内含价值的比例。经贴现率统一调整后：平安VIF/EV≈63.7%、太保≈61.6%、国寿≈41%、新华≈45%。VIF占比越高，内含价值的"含金量"越高——因为VIF按11%年增长远高于净资产按3.5%增长。
- **结论**：国寿的"1元内含价值"和太保的"1元内含价值"含金量不同。一个41%是快速释放的VIF，一个是61.6%。前者对应的是"注水的有效业务价值"——利润大量锁定在遥远未来，远低于11%的理论释放速度。这解释了为什么国寿PEV低但市场仍不买账。
- **结果**：PEV要看分母EV的内部构成。VIF/EV<45%的公司，PEV应打6~7折。

### 案例 3：2016年四家公司集体下调投资收益率假设 — 假设变动对PEV的一次性冲击（第8章/c18）

- **问题**：2016年四家公司大规模下调永续投资收益率假设如何影响估值判断？
- **方法论的使用**：作者整理4家公司假设调整前后对比：平安5.5%→5%（EV一次性损失421亿）、国寿5.5%→5%（同步将贴现率从11%→10%对冲EV冲击）、太保5.2%→5%、新华5.2%→5%。这次"假设地震"使各公司内含价值经历了一次性缩水。
- **结论**：2016年假设调整后的PEV突然"变贵了"，不是因为股价涨了，而是分母EV缩水了。投资者必须区分"假设驱动的PEV变化"和"经营驱动的PEV变化"。国寿通过同步调降贴现率对冲了EV缩水——这是精算"魔术"，不是经营改善。
- **结果**：当前十年期国债已从2016年的约3%进一步降至2025年的约1.6%，远低于内含价值5%的永续假设。若再次集体下调投资收益率假设，所有公司PEV将再次被动上升（EV缩水）。这是当前保险股估值最大的"灰犀牛"。

### 案例 4：新华保险万峰转型 — PEV低估是机会还是陷阱？（第8章/第2章/c08）

- **问题**：新华保险2014-2018年间新单保费腰斩、PEV处于低位，是价值陷阱还是价值发现？
- **方法论的使用**：作者追踪新华在万峰治下的数据：新单保费从441亿降至264亿（主动砍掉低价值趸交和银保），但NBV从49亿升至122亿（+150%），NBV率从11%升至46%。新业务价值的量质齐升证明了转型有效。
- **结论**：这是一个典型的"PEV低估+NBV率上升"=机会的案例。虽然PEV低是因为市场悲观于保费规模萎缩，但NBV率的飙升说明公司的"增长含金量"在提升。当NBV增速由负转正时，PEV从低估向合理修复的戴维斯双击就会发生。
- **结果**：该案例确立了"PEV低估×NBV质量改善"这一最有力的买入信号。但万峰离任后继任者推翻转型策略——也说明了"硬着陆转型"的组织风险。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **保险股买入决策**：用户问"XX保险现在能买吗？PEV 0.6是不是低估了？"→ 需要综合PEV+ROEV+NBV+股息率+保障/储蓄属性做多维度验证
2. **保险股持仓评估**：用户持有某保险股，问"现在该继续持有还是卖出？PEV已经1.3了"→ 需要判断当前PEV是否匹配公司的ROEV和NBV增长前景
3. **跨公司选择**：用户问"中国平安和中国太保哪个更值得投？"→ 需要统一假设口径后，从PEV+EV质量+增长质量+保障属性四维对比
4. **利率环境变化下的估值重估**：用户问"利率降到1.6%了，保险股还能拿吗？"→ 需要评估投资收益率假设下调对EV的冲击和PEV的被动变化
5. **判断"价值陷阱"vs"低估机会"**：用户问"XX保险跌了50%了，是不是已经到底了？"→ 需要区分NBV负增长是周期性的还是结构性的，PEV低是因为恐慌还是因为价值毁灭
6. **分红回报评估**：用户问"XX保险股息率5%，能不能当收息股？"→ 需要判断股息率是否可持续，PEV是否有下行风险吞噬股息
7. **政策/准则变动影响评估**：用户问"偿二代二期对保险股估值有什么影响？""IFRS17后怎么看保险股估值？"→ 需要更新假设参数，重新校准PEV基准

### 与相邻 skill 的区分

- 与 **ev-valuation**：本skill是估值决策的**最终裁判**（综合PEV+ROEV+NBV+股息率+属性→买入/持有/卖出建议），ev-valuation是**EV维度的深入拆解**（调整净资产+VIF+CoC的内部构造和敏感性）。本skill调用ev-valuation的输出作为决策输入之一。
- 与 **nbv-analysis**：本skill使用NBV分析结论来判断PEV溢价/折价的合理性（NBV增速>20%可以接受高PEV），但不负责NBV本身的拆解。NBV拆解分析是nbv-analysis的职责。
- 与 **op-four-factors**：本skill使用营运利润四因子拆解来判断股息率的可持续性（剩余边际摊销增速→营运利润增速→分红增速），但不负责OP的拆解过程。
- 与 **triple-framework**：本skill在估值判断前，默认用户已理解三套准则世界观（或先调用triple-framework建立认知基础）。三套准则框架是估值判断的先决认知。
- 与 **apple-tree-model**：苹果树模型是理解"为什么保险股不能用PE估值"的入门故事，本skill是解决"那到底该怎么估值"的操作答案。

## E — 可执行步骤 (Execution)

### 阶段 0：确认输入与数据范围

向用户确认或收集以下信息：
1. **目标保险公司名称**（人身险/寿险公司，如中国平安、中国人寿、中国太保、新华保险）
2. **报告期**（如"2025Q1"或"2024年报"）
3. **数据来源**：用户直接提供关键指标，或指定财报PDF/TXT路径

若用户仅提供公司名称而未给数据，需说明"需要以下核心数据才能完成估值判断"，列出所需指标清单，等待用户提供。

### 阶段 1：计算PEV — 主锚定位

**执行步骤**：
1. 获取当前股价和最新披露的每股内含价值（从年报"内含价值分析"章节）
2. 计算 `PEV = 股价 ÷ 每股内含价值`
3. 获取历史PEV区间（近3-5年最高/最低/中位数），判断当前处于历史分位
4. 获取同业可比公司的PEV（统一贴现率口径后），横向量化相对贵贱

**贴现率口径统一**（关键步骤）：
- 确认各公司的风险贴现率：平安11%、太保11%、国寿10%、新华11.5%（以2019年为基准，需核实最新）
- 若比较国寿（10%贴现率），将其内含价值按11%贴现率重估：国寿EV假设上调至11%后，EV约缩水5%~8%，PEV相应上升
- 同理，新华（11.5%）相对保守，PEV被低估

**完成标准**：输出"当前PEV = X.XX，处于历史Y%分位，同业排第Z位，经贴现率统一调整后PEV_adjusted = X.XX"

**判停条件**：
- PEV < 0.5 且 NBV连续2年负增长 → 不是"低估"，先诊断是否为价值陷阱。进入阶段2深入验证。
- 公司未披露内含价值 → 告知用户"该公司不适用内含价值估值体系"，停止分析。

### 阶段 2：检验EV质量 — 分母里的含金量

调用 `ev-valuation` 的核心分析，重点完成：

1. **VIF/EV占比**：有效业务价值 ÷ 内含价值。>55%为优质（保障型），<45%为注水（储蓄型）。
2. **VIF实际释放率**：对比"期初VIF×11%"与"有效业务价值当期释放的实际利润"。若实际释放远低于11%，确认"注水VIF"。
3. **投资收益率假设压力测试**：当前十年期国债约1.6%，而内含价值假设永续5%。若假设下调50bp至4.5%，EV缩水幅度？
4. **贴现率敏感性**：贴现率每±0.5%，NBV变动幅度，反向估算EV变动。

**完成标准**：回答"每1元内含价值，其中X元是快速释放的VIF，Y元是慢速释放的注水VIF，Z元是净资产。在利率下行环境下，EV面临X%的下调风险。"

### 阶段 3：评估NBV增长前景 — 分子里的发动机

调用 `nbv-analysis` 的核心分析：

1. **NBV增速趋势**：近3-5年扣偿后NBV增速，判断是加速/减速/负增长
2. **NBV增速驱动力**：来自量（新单保费扩张）还是质（NBV率提升）？
3. **人均NBV趋势**：代理人人均NBV是在上升还是下降？（上升=内生增长，下降=吃老本）
4. **渠道结构**：代理人渠道新单占比是在提升还是下降？（提升=质量改善）

**完成标准**：回答"未来1-3年NBV增速大概率在X%~Y%区间，驱动力主要来自Z（量/质/渠道/假设），可持续性判断为A（可持续/拐点/恶化）"

### 阶段 4：计算ROEV — 内生回报率校准

用近似公式计算ROEV：

```
ROEV ≈ (期初调整净资产 × 3.5% + 期初有效业务价值 × 11% + 扣偿后NBV) ÷ 期初内含价值
```

或直接取自内含价值变动表中的"内含价值增长率"扣除非经营因素（假设变动、市场价值调整等）。

**完成标准**：回答"该公司ROEV约为X%~Y%，属于Z型（债券型/正常成长型/高速成长型）"

### 阶段 5：确定保障型 vs 储蓄型属性 — 估值逻辑分支

通过以下指标综合判定公司属性：

| 判定指标 | 保障型特征 | 储蓄型特征 |
|---------|-----------|-----------|
| NBV率 | > 35% | < 30% |
| VIF/EV占比 | > 55% | < 45% |
| 代理人渠道新单占比 | > 65% | < 55% |
| 利差依赖度 | 低（死差/费差为主） | 高（利差为主） |
| 产品结构 | 长期健康险/定期寿险为主 | 分红险/万能险/年金险为主 |

**完成标准**：明确判定"该公司属于保障型/储蓄型/混合型"，据此选择估值基准区间。

### 阶段 6：评估股息率 — 落袋为安的地板

1. 获取当前股息率（近12个月每股分红 ÷ 当前股价）
2. 确认分红基础：
   - 以**营运利润为分红基础**（如中国平安）→ 分红增长可预期
   - 以**净利润为分红基础**（多数央企）→ 分红波动大
3. 结合剩余边际蓄水池健康度（调用 `residual-margin-analysis`），判断分红可持续性：
   - 剩余边际进水/出水比 > 1.2 → 分红增速可维持
   - 进水/出水比 < 1.0 → 分红有下调风险

**完成标准**：回答"股息率X%，分红基础为Y，可持续性为Z。对应当前PEV，股息回报吸引力为A（足够/勉强/不足）"

### 阶段 7：综合估值决策矩阵输出

将以上5步结果填入决策矩阵，得出最终判断：

```
┌─────────┬──────────┬──────────┬──────────┬──────────┬──────────┬────────────┐
│   PEV   │   ROEV   │ NBV增速  │  属性    │  股息率  │  历史分位 │  综合判断  │
├─────────┼──────────┼──────────┼──────────┼──────────┼──────────┼────────────┤
│  X.XX   │  XX%     │  XX%     │ 保障/储蓄│  X.X%    │  X%分位  │ 买入/持有/ │
│         │          │          │          │          │          │ 卖出/观望  │
└─────────┴──────────┴──────────┴──────────┴──────────┴──────────┴────────────┘
```

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **分析产险/财产险公司**（如人保财险、平安产险）：产险是短期业务，不存在内含价值概念。产险应用PB+ROE或综合成本率(COR)估值
- **仅查询单一指标**：如用户只问"平安PEV多少"或"国寿NBV增长率"，直接给出数字即可，无需激活全套估值框架
- **宏观利率趋势讨论（无公司标的）**："利率下行保险股怎么走"→ 这是宏观分析，不是公司估值
- **首次接触保险股的认知入门**：用户问"为什么保险股PE这么低"→ 应先用apple-tree-model建立认知基础，而非直接用PEV做判断
- **非上市保险公司的估值**：未披露内含价值的公司，只能用PB/ROE或可比公司法，不在本skill覆盖范围
- **纯技术面/择时决策**："保险股跌到20日均线了要不要抄底"→ 这是技术分析，非基本面估值

### 作者在书中警告的失败模式

- **"PEV低=低估"的机械判断**：不检查EV质量就下结论是危险的。若VIF/EV占比仅40%且VIF注水严重，PEV 0.6也不是真正的低估——因为"1元内含价值"里只有4毛是快速释放的有效价值，另外6毛是被锁定在遥远未来的"纸上富贵"。
- **不统一贴现率口径就做跨公司PEV对比**：国寿10%贴现率报出的PEV含"贴现率红利"，若统一至11%，PEV将上升5%~8%。直接拿年报PEV排大小的排名可能完全颠倒。
- **5%永续投资收益率假设的"皇帝新衣"**：当十年期国债已降至1.6%，内含价值的5%永续投资收益率假设是否还能维持？若未来集体下调至4%~4.5%，所有保险公司的PEV将被动上升（EV缩水）。在利率下行周期中买入保险股，必须把"假设下调"这个灰犀牛计入安全边际。
- **将NBV贴现率下调误读为经营改善**：若某公司将贴现率从11%调降至10.5%，NBV会因此虚增约5%。这不是经营改善，是参数游戏。估值判断时必须剔除假设变动的影响。
- **忽略分红政策对投资回报的影响**：PEV相同的两家公司，若一家以营运利润为分红基础（分红率60%），另一家以净利润为分红基础（分红率30%），投资者的实际到手回报差一倍。只看PEV不看分红政策，等于只看房价不看租金。

### 作者的盲点 / 时代局限

- **2020年后超低利率环境**：本书成书时十年期国债约3.1%，PEV的估值基准（1.0为合理中枢）是在那个利率环境下建立的。2025年十年期国债约1.6%，整个估值锚可能需要系统性下移——"合理PEV"可能从1.0降至0.7~0.8。在使用本skill时，应结合最新利率环境调整PEV基准。
- **偿二代二期（2022年）的影响**：偿二代二期对实际资本认定收紧，影响最低资本计算→持偿成本→内含价值。各公司EV在偿二代二期下可能再次发生变化，本书未覆盖。
- **IFRS17（2023年生效）**：新准则改变了保险合同的收入确认方式，虽然内含价值体系本身是精算概念不受会计准则直接影响，但保险公司对外披露的格式和指标颗粒度正在变化，可能影响数据可得性。
- **代理人大幅萎缩（2020-2024年）**：本书写作时行业代理人约900万，至2024年已降至不足300万。NBV增速的"量"驱动因子（人力扩张）已全面失效，分析框架需更侧重"质"（人均NBV提升对抗人力下降）。
- **作者以平安为最优解标杆**：书中跨公司对比的结论天然偏向保障型公司（平安/太保），对储蓄型公司的估值讨论不够深入。在实际运用中，储蓄型公司（国寿/新华）的估值逻辑应独立判断，不能简单以平安为基准比较。

### 容易混淆的邻近方法论

- **混淆PEV和PB**：PEV=股价/内含价值，PB=股价/账面净资产。保险股的PB可能1.0而PEV仅0.6（因为EV远大于净资产），用PB判断保险股估值是看错了分母。但要注意——两者的安全边际含义不同：PEV的安全边际在EV假设中，PB的安全边际在清算价值中。
- **混淆ROEV和ROE**：保险公司的会计ROE因"费用前置利润后置"而失真（高增长期ROE偏低），ROEV是基于内含价值的真实回报率，更能反映价值创造能力。一个ROE仅8%但ROEV达15%的保险公司，可能是高价值的成长股。
- **混淆NBV率和净利润率**：NBV率动辄40%以上，但实际上真正的利润率（NBV/保费折现值）仅约10%。不能用NBV率来替代净利润率的判断。
- **将保险股估值与银行股估值直接对标**：银行用PB+ROE或DDM，保险用PEV+ROEV+NBV。两者的"净资产"含金量完全不同（保险的净资产里藏着巨量剩余边际未释放），不能看到"保险PB<1"就说"比银行还便宜"。

## 原子 skill 编排

本聚合 skill 在估值决策过程中调用以下原子 skill 的方法论（各原子skill的详细方法论见各自的SKILL.md）。原子 skill 不面向用户直接调用。

```
估值决策流水线
│
├─ 阶段1: PEV计算 ← 直接计算，无需调用原子skill
│
├─ 阶段2: EV质量检验 ← 调用 ev-valuation 的核心方法
│   └─ VIF/EV占比 + VIF释放率 + 投资假设敏感性 + 贴现率敏感性
│
├─ 阶段3: NBV增长评估 ← 调用 nbv-analysis 的核心方法
│   └─ NBV增速 + NBV率 + 人均NBV + 渠道结构 + 参数统一口径
│
├─ 阶段4: ROEV计算 ← 基于ev-valuation的公式 + 内含价值变动表
│
├─ 阶段5: 保障/储蓄判定 ← 综合阶段2+阶段3的指标
│
├─ 阶段6: 股息评估 ← 调用 op-four-factors + residual-margin-analysis
│   └─ 分红基础 + 剩余边际蓄水池健康度 → 分红可持续性
│
└─ 阶段7-8: 综合决策 + 输出报告
```
