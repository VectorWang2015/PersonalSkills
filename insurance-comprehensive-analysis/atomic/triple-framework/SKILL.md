# 三套准则世界观框架 — GAAP / EV / OP 的钩稽关系

## R — 原文 (Reading)

> 一般会计准则的出发点是"从时间的起点去还原现在"……内含价值评估标准，颇有"从时间的终点来还原现在"的味道……
> 三套准则就像三种看世界的价值观，也像人身险公司在三个世界的投影，相互纠缠，绚丽多姿。
>

> 其安身立命的核心便是两边讨好的能力。
>

> 但从某种程度而言，分析保险公司又是简单的，因为只要理解了三套准则之间的逻辑，

## I — 方法论骨架 (Interpretation)

### 核心命题

保险公司（尤其人身险公司）因为"费用前置、利润后置"的苹果树商业模式，使得单一准则下的财务数据必然失真。
投资者必须同时理解三套准则及其相互钩稽关系，才能穿透表面数据看到真实价值。

### 三套准则的时间箭头

| 准则 | 时间箭头 | 核心问题 | 适用场景 | 角色隐喻 |
|------|----------|----------|----------|----------|
| **GAAP**（一般会计准则） | 过去 → 现在 | "已经发生了什么？" | 排雷、筛查财务造假 | 钢铁直男 |
| **EV**（内含价值评估标准） | 未来 → 现在 | "将来能赚多少？" | 估值、判断贵贱 | 拉普拉斯恶魔 |
| **OP**（营运利润体系） | 嫁接 GAAP 和 EV | "可预期的真实利润是多少？" | 跟踪经营质量、分红基础 | 现实主义商人 |

### 三套准则的钩稽关系（推背图全息映射）

每一套准则下，保险公司的经济活动都是"**销售 → 营运 → 投资**"三条主线，但三条主线在不同准则下的
**"投影"（映射科目）** 各不相同。理解这种映射关系是三套准则框架的核心：

```
                        销售                        营运                        投资
                        --                          --                          --
   GAAP（直男视角）     新单保费拆解                三项费用/已赚保费比         总投资收益率
                        人均产能×规模÷占比          四大名捕成本结构(赔付/       净投资收益率
                                                    红利/准备金/退保)
                        
   EV  （先知视角）     扣偿后新业务价值(NBV)       有效业务价值(VIF)          投资收益率假设(5%)
                        NBV增长驱动因子拆解          营运经验差异                投资回报差异
                                                    模型及假设变动              (实际vs假设的差异)
                        
   OP  （商人视角）     新业务贡献的剩余边际        剩余边际摊销               净资产投资收益(按5%)
                        (剩余边际蓄水池进水口)       营运偏差                   息差收入
```

### 钩稽规则（准则间的关键桥梁）

1. **剩余边际是连接三套准则的金线**：
   - 在 GAAP 中，剩余边际是准备金的核心组成部分（"利润蓄水池"），承保时锁入，逐年摊销释放
   - 在 OP 中，剩余边际摊销是营运利润的最大来源（占比约66%），是"可预期利润"的基石
   - 在 EV 中，剩余边际通过"税收和资本差异"+"贴现率等经济假设差异"+"其他差异"三座桥梁，投影为有效业务价值

2. **5% 投资收益率假设是两套准则的公共锚**：
   - EV 体系中，有效业务价值和调整净资产均以 5% 永续投资收益率作为核心假设
   - OP 体系中，将实际投资收益中超过 5% 的部分剔除，统一按 5% 计入，使利润"可预期"
   - 实际投资与 5% 的差额通过 EV 的"投资回报差异"逐年追回——都是"锅里的肉，一块也跑不了"

3. **新业务价值(NBV)是三套准则的增长发动机**：
   - 在 EV 中：NBV 在第二年转化为有效业务价值，驱动内含价值增长
   - 在 OP 中：NBV 贡献的剩余边际进入蓄水池，逐年摊销释放为营运利润
   - 在 GAAP 中：新单保费本身不反映真实盈利能力，但种下"未来的苹果"

4. **营运偏差是 EV 与 OP 的交叉验证点**：
   - EV 中的"营运经验差异"和 OP 中的"营运偏差"指向同一事物（实际费用低于精算假设）
   - 但如果方向不一致，可能意味着公司在用"费用率假设"操纵内含价值（费用假设越保守→EV越低但偏差越正）

### 使用三套准则的统一心智模型

```
第一步：用 GAAP 排雷
  → 排除四大名捕异常、费用/保费比过高、退保率激增的公司
  → GAAP不好大概率真有问题，GAAP好不一定真好

第二步：用 EV 判断估值
  → P/EV 跨公司比较（统一贴现率口径后）
  → VIF/EV 占比判断内含价值质量（高占比=高增速=高质量）
  → 新业务价值率/增长趋势判断成长性

第三步：用 OP 跟踪经营质量
  → 营运利润增长是否稳健（排除短期投资波动和折现率变动后）
  → 四因子结构是否健康（剩余边际摊销占比应稳定或上升）
  → 营运利润是否被作为分红基数（平安是，太保在推进）

第四步：交叉验证
  → EV的投资回报差异是否持续为正（验证5%假设的保守性）
  → OP的剩余边际摊销增速 vs EV的有效业务价值释放增速是否匹配
  → GAAP的净利润剔除调整项后，是否与OP趋势一致
```

## A1 — 书中的应用 (Past Application)

### 案例 1：平安 vs 太保 — 同一组数据在三套准则下的三重解读

- **场景**：中国平安 2019 年年报数据，作者演示同一家公司在不同准则值域下的分析路径
- **GAAP 视角**：净利润 1494 亿元，PE 仅约 10 倍——看似便宜，但这是被"费用前置利润后置"低估后的表象。新业务越猛的公司越被低估
- **EV 视角**：内含价值 1.2 万亿元，P/EV 约 1.3 倍——实际上并不便宜。内含价值已将未来所有保单利润折现回来，体现了公司"真正值多少钱"
- **OP 视角**：营运利润 1327 亿元（剔除 167 亿短期投资波动和其他一次性项目后），比净利润的波动性大幅降低，且被作为分红基数——"可预期的真实利润"
- **结论**：同一家公司，用 PE 看很便宜、用 P/EV 看合理、用 OP 看分红可持续。单一准则下做投资决策必然偏差，三套兼修才能避免"被低估骗进去"

### 案例 2：苹果树故事 — 三套准则同一故事的不同讲述

- **场景**：作者虚构农夫种苹果树的故事，用同一商业模式在三套准则下讲出三种截然不同的故事版本
- **GAAP 版本**：种树费用前置 → 第一年账面亏损（维修准备金一年计提） → 农场越种越亏 → 用 PB 估值会发现"越扩张越便宜"，实为陷阱
- **EV 版本**：站在果树生命终点 → 将未来所有苹果的利润按 11% 贴现回来 → 农场价值远高于账面 → "种得越多价值越高"，但完全依赖假设准确性
- **OP 版本**：剔除短期波动 → 将维修费用在全生命周期摊销 → 农场显示为稳健的正利润 → 更接地气但仍在"讨价还价"
- **关键启示**：GAAP 系统性低估"扩张期"保险公司 → 越是高成长的人身险公司，GAAP 越失真 → 必须用 EV 补充、用 OP 验证

### 案例 3：国寿 — "有效业务价值高但利润释放慢"的钩稽诊断

- **场景**：中国人寿内含价值看起来不低，但实际利润释放偏慢，GAAP 净利润长期低迷
- **问题**：用三套准则交叉诊断。GAAP 利润差 → EV 中 VIF 高 → 为什么利润释放不出来？
- **诊断过程**：VIF/EV 占比仅 ~41%（远低于平安的 63.7%） → 有效业务价值贴现率 11% 的预计释放在当期不等于 11% → 中国人寿的产品以分红型和万能型为主，利润大量锁定在保单终末期释放 → 属于"注水的有效业务价值" → 即使 EV 数字不错，到 GAAP 和 OP 的传导链条过长
- **结论**：三套准则之间的"传导效率"是判断保险公司价值的隐藏维度。VIF/EV 占比 + VIF 实际释放率 才是 EV 质量的核心判据，而非只看 EV 数字大小。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. **面对保险股感到"分裂"**：PE 很低（8倍）、P/EV 却合理（1.5倍），不知道到底便不便宜
2. **GAAP 利润和 EV/OP 严重矛盾**：净利润同比大幅下降但营运利润稳定增长、或内含价值平稳但净利润忽上忽下
3. **首次接触保险股分析**：从银行/消费行业转过来，试图用 PE/PB 估值保险股，需要理解为什么那些方法不适用
4. **跨公司对比混乱**：同一行业，平安净资产收益率高但新华 P/EV 更低，不知如何统一比较
5. **分红可持续性判断**：公司宣布分红与营运利润挂钩，需要理解营运利润的"可预期性"究竟来自哪里
6. **偿付能力/精算假设变动解读**：偿二代二期收紧、投资收益率假设调整、贴现率变化——需要理解这些变动会同时影响哪些准则框架

### 语言信号（用户的话里出现这些就应激活）

- "保险公司的 PE 为什么这么低，是不是被低估了？"
- "这家公司净利润大跌但内含价值还在增长，怎么回事？"
- "怎么看保险股？PE还是PB？"
- "剩余边际是什么？和净利润有什么关系？"
- "内含价值和营运利润，到底哪个更重要？"
- "为什么中国平安用 P/EV，国寿用 P/PEV 但结论不一样？"
- "保险公司的 ROE 为什么这么不稳定？"

### 与相邻 skill 的区分

- 与 **内含价值估值框架** 的区别：本 skill 是三套准则的**关系地图**（为什么需要三套、如何配合使用），内含价值 skill 是 EV 体系的**纵深拆解**（ANA+VIF+CoC 的内部构造和驱动因子）。在分析流程中，本 skill 先建立世界观坐标系，内含价值 skill 再在其中纵深展开。
- 与 **营运利润四因子分解** 的区别：本 skill 解释 OP 在三套准则中的**角色和定位**（为什么诞生、嫁接什么），营运利润 skill 是 OP 内部的**成分拆解**（四个因子各自的驱动和变化趋势）。
- 与 **GAAP 排雷原则** 的区别：本 skill 规定 GAAP 在整体分析中的**功能边界**（只能排雷不能估值），GAAP 排雷 skill 是该功能的具体**执行方法论**（怎么排、排什么、怎么计算多年平均值）。
- 与 **苹果树抽象模型** 的区别：苹果树模型是三套准则框架的**叙事化呈现**（用故事讲概念），本 skill 是三套准则框架的**分析化呈现**（用钩稽表做分析）。前者是"入门理解"，后者是"操作手册"。

## E — 可执行步骤 (Execution)

### 步骤 1：建立三套准则坐标系 — 绘制"推背图"映射表

从目标保险公司年报中提取以下数据，按三行（GAAP/EV/OP）×三列（销售/营运/投资）组织成 3×3 表格：

|  | 销售（获客） | 营运（存量） | 投资（资产） |
|--|-------------|-------------|-------------|
| **GAAP** | 新单保费（人均产能×规模人力÷代理人新单占比） | 三项费用/已赚保费比；四大名捕成本结构 | 净投资收益率；总投资收益率 |
| **EV** | 扣偿后新业务价值(NBV)；NBV率 | 有效业务价值(VIF)/EV占比；营运经验差异 | 投资收益率假设(5%)；投资回报差异 |
| **OP** | 新业务贡献的剩余边际 | 剩余边际摊销；营运偏差 | 净资产投资收益(按5%)；息差收入 |

**完成标准**：产出至少 2 期（当前+去年同期）3×3 映射表，可直接观察各准则下各主线的变化方向。

**判停条件**：如果目标公司仅披露 GAAP 数据，未披露 EV 和 OP 数据 → 告知用户"该公司仅在一套准则下可见，无法形成三套交叉验证"，不强行推演缺失维度。

### 步骤 2：用 GAAP 排雷 — 排除财报基础面的问题

执行简单的 GAAP 筛查（详细方法论见 `atomic/gaap-screening/SKILL.md`）：
1. **计算 6 年平均值**（修正"费用前置利润后置"的单期失真）：各项指标取 6 年算术平均
2. **四大名捕结构**：赔付+分红 / 提取准备金 / 退保金 占比是否稳定
3. **三项费用/已赚保费**：是否持续偏高（>同业均值）
4. **退保率趋势**：是否加速上升（>6% 需警惕"长险短做"）

**完成标准**：回答"这家公司在 GAAP 下存在严重的经营问题吗？"→ 如果 YES，分析在此终止，直接输出排雷结论。

### 步骤 3：用 EV 判断估值 — 统一贴现率口径后横向对比

1. **P/EV 计算**：当前市值 ÷ 最新内含价值
2. **贴现率口径统一**（关键步骤）：
   - 平安使用 11%，其他公司可能使用不同贴现率
   - 简单调整：P/EV_adjusted = P/EV × (对方贴现率 / 平安贴现率)
   - 更精确的方法见 `atomic/embedded-value-decomposition/SKILL.md`
3. **EV 质量评估**：
   - VIF/EV 占比（越高越好，平安63.7%为标杆）— 但必须注意"注水VIF"
   - VIF 实际释放率 vs 理论的 11%（差距越大越"注水"）
   - 分散效应贡献（13-16% 是正常范围）
4. **增长质量评估**：
   - NBV 增速是否为正
   - NBV 率是否稳定（剔除一次性影响后）
   - 投资回报差异累进方向（持续为正 → 假设保守；持续为负 → 假设激进）

**完成标准**：回答"按 EV 标准，这家公司的估值是贵了还是便宜了？内含价值质量高还是注水了？"

### 步骤 4：用 OP 验证经营质量 — 利润"可预期性"检验

1. **营运利润趋势**（至少 3 年）：是否稳定增长
2. **四因子结构**（详见 `atomic/operating-profit-decomposition/SKILL.md`）：
   - 剩余边际摊销占比是否稳定（约66%为健康水平）
   - 营运偏差是否持续为正（持续正偏差 = 精算假设保守 = 管理层可信）
   - 净资产投资收益和息差收入趋势
3. **调整项目的合理性检验**：
   - 短期投资波动剔除额 / 总投资收益 → 是否合理（平安长期~70%）
   - 折现率变动影响的方向和幅度
   - 一次性项目是否真的"一次性"

**完成标准**：回答"这家公司的可预期利润增长是否稳健？分红可持续吗？"

### 步骤 5：三套准则交叉验证 — 揭示隐藏矛盾

将步骤 2-4 的关键指标进行跨准则对照：

| 交叉验证点 | GAAP 信号 | EV 信号 | OP 信号 | 矛盾判断 |

| 保守性 | 准备金提取率 | 投资回报差异累进 + 营运经验差异 | 营运偏差方向 | GAAP准备金提取高+EV/OP偏差持续正=极度保守(安全)；反之=激进(危险) |
| 增长性 | 新单保费增速 | NBV增速 | 新业务贡献剩余边际增速 | NBV增速<新单保费增速=产品利润率收缩；新业务贡献<NBV=激进假设 |

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- **非人身险保险公司**：本框架专为人身险公司（寿险/健康险/养老险）设计，产险公司不适用"费用前置利润后置"模式，内含价值评估标准对其也意义有限
- **互联网保险新物种的单一评估**：如众安在线，其"科技+保险"双元模式需要在三套准则框架之上叠加保险科技三阶段演化模型
- **单一指标快速查询**：如仅问"平安新业务价值多少"，直接用数字回答即可，无需激活整套框架
- **纯宏观利率环境讨论**：分析利率下行对保险行业的影响属于宏观分析，本 skill 聚焦公司层面的准则框架映射
- **仅看一期季报就做结论**：三套准则框架的分析至少需要年度数据 + 至少 3 年趋势。季报数据不完整（多数公司不披露季度 EV/OP），强行分析会导致盲人摸象

### 作者在书中警告的失败模式

- **用 PE/PB 估值人身险公司**（ce 多次强调）：PE 被"费用前置利润后置"系统性压低，成交量越大的公司 PE 越失真。用 PE 看保险股就像用磅秤称羽毛——不是工具坏了，是工具不适用
- **只看 EV 数字不看质量**（ce 第五章）：VIF 占比低（<45%）或实际释放率远小于理论的 11%（"注水VIF"），即使 P/EV 看起来低也不是真正的便宜
- **混淆 NBV 率和利润率**（ce 第五章/p17）：新业务价值率 = NBV / 首年保费 ≠ 利润率（时间维度不一致）。真正利润率 = NBV / 保费折现值，典型值约 10%，而非注水后的 60%
- **假设调整当一次性事件无视**（ce 第五章）：投资收益率假设或贴现率调整会同时影响 EV（直接改变）、OP（通过剩余边际摊销传导）、GAAP（通过准备金计提传导），三套准则同时受影响
- **GAAP 好就认为公司好**（ce 第二章/p04）：GAAP 的利润可以通过削减销售费用、操纵退保金等方式短期美化。GAAP 好不一定真好，但 GAAP 极差大概率真有问题（"公众场合都缺德的人私底下大概率不行"）

### 作者的盲点 / 时代局限

- **IFRS17 新准则（2023年生效）**：本书写作于 2021 年前，未覆盖 IFRS17 对保险会计的重大重塑。IFRS17 将改变准备金的分拆方式和利润释放模式，但作者的三套准则世界观框架（时间箭头思维）反而因此更加重要——IFRS17 只是让 GAAP 更接近 EV 逻辑，并不改变"多准则交叉验证"的必要性
- **偿二代二期（2022年）**：对实际资本的认定收紧，影响偿付能力充足率 → 持偿成本 → 内含价值。三套准则框架本身不变，但参数需更新
- **低利率时代的 5% 假设压力**：作者写作时市场利率尚可维持 5% 长期投资收益率假设，但后疫情低利率环境（10Y国债2.5%~3%）使这一假设面临挑战。5% 永续假设若在未来被系统性调低，三套准则的 5% 公共锚将同步偏移，需整体重新校准
- **数据依赖中国平安**：作者以平安为主要标杆，其披露的营运利润体系最完整（四因子有明细拆解），其他公司（国寿/太保/新华）的 OP 披露深度逐级递减。对非平安公司的分析精度会因数据可得性而降低

### 容易混淆的邻近方法论

- **与内含价值估值框架**：本 skill 告诉你"为什么要同时看三套、如何分工"，内含价值 skill 告诉你"EV 具体怎么算、怎么拆"。先学本 skill 建立坐标系，再深入 EV skill 学细节
- **与苹果树抽象模型**：同一套框架的两种表达——苹果树用故事入门，本 skill 用表格操作。实战中先回忆苹果树的故事化直觉，再用本 skill 的 3×3 表格做系统分析
- **与推背图全息分析框架**：推背图是本 skill 所描述映射关系的**可视化全图版**（一张图囊括所有映射），本 skill 是**可执行步骤版**（分步骤执行检查）。两者共享同一底层知识，分工不同
