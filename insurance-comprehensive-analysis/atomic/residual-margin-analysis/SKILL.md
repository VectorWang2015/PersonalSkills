# 剩余边际分析框架

## R — 原文 (Reading)

> 剩余边际是首日利得。我们不应该在首日确认利得，而是将其作为剩余边际，计入保险合同准备金。首日利得可以这样计算：可获取的保费现值 - 承保那一刻预计的赔付、费用等所有支出之和。剩余边际犹如水池里看不到的部分，是准备金里的"宝藏"。
>
> — 东先生，《读懂保险股》第3章

> 如果我们把剩余边际的余额想象为一个蓄水池，出水口即是剩余边际的摊销，进水口则是每年承保的新单中蕴含的利润。出水口越大，会计利润和营运利润释放越多；进水口越大，未来的利润释放越有保障。
>
> — 东先生，《读懂保险股》第6章

---

## I — 方法论骨架 (Interpretation)

剩余边际分析的底层逻辑是：**保险公司的真实盈利能力被一般会计准则系统性隐藏，而剩余边际就是那个"藏利润"的容器**。

具体分为四个递进层次：

**第一层：理解剩余边际的"出身"**。人身险公司承保时（合同签署那一刻），精算师计算出这张保单未来生命周期中能赚多少钱。按照会计准则，这笔"首日利得"不能立刻计入利润，而是被强制锁入准备金的一个子科目——剩余边际。也就是说，保险公司在签单那一刻就已经赚到了利润，只是会计上不让你看到。

**第二层：理解会计准备金的三层结构**。会计准备金 = 合理估计负债（BEL，中性的最优估计）+ 风险边际（保守的安全垫）+ 剩余边际（利润暂存库）。三层结构的关键区分——BEL 和风险边际是对未来不确定性的应对，而剩余边际本质上已经是利润了，只是"还没到释放的时间"。剩余边际在准备金的"最底层"，是保险公司审慎性的终极体现。

**第三层：掌握蓄水池模型（进水-存量-出水）**。将剩余边际余额视为一个动态蓄水池：
- **进水口**：每年新单承保时锁入的剩余边际（新业务贡献）+ 存量余额按约5%利率计算的预期利息增长
- **存量**：期末剩余边际余额（资产负债表上的"隐藏利润"总量，可能超过净资产）
- **出水口**：剩余边际摊销（按发单时锁定的摊销模式逐期释放，不受资本市场波动影响）

核心洞察：出水口/总蓄水池的比例（约9.5%，平安2019年数据）反映利润释放速度；进水口增速反映新业务质量。

**第四层：串联至营运利润体系**。营运利润 = 剩余边际摊销 + 净资产投资收益 + 息差收入 + 营运偏差。其中剩余边际摊销占比约66-74%（平安2019年为744亿/1005亿=74%），是营运利润的绝对主体。换言之，保险公司的"可预期利润"大部分来自过去已赚、现在释放的剩余边际，而非当期新赚的钱。

## A1 — 书中的应用 (Past Application)

### 案例 1: 中国平安2019年剩余边际蓄水池全景拆解

- **问题**: 中国平安隐藏了多少利润？剩余边际的进出水是否健康？
- **方法论的使用**: 作者以平安2019年年报第59页为素材，拆解剩余边际的蓄水池全貌。进水口：新业务价值贡献1557亿元；出水口：剩余边际摊销744亿元；净增813亿元；期末存量：9184亿元。作者进一步计算摊销比例：744亿/期初余额（8371亿）≈ 8.9%，反映历史摊销速度；进水/出水比=1557/744≈2.1倍，反映新业务仍在强劲地为蓄水池"注水"。
- **结论**: 平安的剩余边际余额9184亿元，比整个集团合并净资产8523亿元还多出661亿元。这意味着——只看普通会计准则下的净资产，你错过了平安超过一半的真实企业价值。而且进水口是出水口的2倍以上，蓄水池仍在扩张，未来利润释放有充分保障。
- **结果**: 剩余边际预计按约20%左右的年复合增速增长，未来将逐年转化为营运利润→会计利润→可分配利润→分红，构成平安内在价值增长的稳定引擎。

### 案例 2: 平安2019年营运利润四来源拆解——剩余边际摊销的主体地位

- **问题**: 平安的营运利润1005亿到底从哪里来？各来源的稳定性如何？
- **方法论的使用**: 作者将平安2019年寿险及健康险业务税前营运利润（1005.45亿元）拆为四个来源：剩余边际摊销744.54亿（74.0%）、净资产投资收益117.38亿（11.7%）、息差收入39.47亿（3.9%）、营运偏差及其他104.06亿（10.4%）。并指出四个来源分别映射三条业务主线：剩余边际摊销→销售端（新业务贡献）、净资产投资收益+息差收入→投资端（资产配置与利差管理）、营运偏差→营运端（实际费用低于精算假设）。
- **结论**: 剩余边际摊销占营运利润的74%，是当之无愧的主体。这意味着营运利润的稳定性和可预测性高度依赖剩余边际摊销的锁定机制——摊销模式在发单时已确定，不受资本市场（股市涨跌、利率波动）影响。净资产投资收益和息差收入虽稳定但也受5%假设约束；营运偏差则受管理层假设保守度影响，存在"跷跷板效应"。
- **结果**: 理解营运利润=理解剩余边际摊销。营运利润增速≈剩余边际摊销增速≈前期新业务价值增速（有滞后效应）。若新业务价值增长放缓，预计2-3年后剩余边际摊销增速也将放缓，最终拖累营运利润→分红→估值。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. **穿透保险公司的"真实盈利"**：用户问"这家保险公司到底赚了多少钱？为什么利润表看不出来？"→需要通过剩余边际的规模、摊销速度和进水口增速来还原被GAAP隐藏的真实盈利水平。
2. **评估保险公司的利润可持续性**：用户问"这家公司的利润能持续吗？会不会某一年突然暴跌？"→分析剩余边际摊销占营运利润的比例，占比越高则利润越稳定（摊销模式在发单时锁定，不受当期市场波动影响）。
3. **预判利润增速拐点**：用户问"这家保险公司的利润增速会不会放缓？"→通过新业务价值增速的变化，预判2-3年后剩余边际摊销增速的走向——新业务价值是先行指标，剩余边际摊销是滞后指标。
4. **识别"利润释放陷阱"**：用户问"这家公司利润很好但代理人一直在减少，正常吗？"→可能是在吃存量蓄水池的老本（出水口仍在释放但进水口在萎缩），识别剩余边际余额是否在下降、进水/出水比是否小于1。
5. **对比不同保险公司的利润质量**：用户问"A和B两家保险公司净利润差不多，哪家更值钱？"→比较剩余边际余额的绝对规模、摊销占比、蓄水池进出水结构——剩余边际余额更大的公司，未来利润释放的厚度更好。

- "剩余边际是什么意思"

- "保险公司为什么利润表看起来不赚钱"
- "营运利润和会计利润差在哪"
- "保险公司真正的赚钱能力怎么分析"

### 与相邻 skill 的区分

- 与 `embedded-value-framework`（内含价值估值）的区别：内含价值是从DCF角度评估保险公司"值多少钱"，剩余边际分析是从准备金角度评估"隐藏了多少利润、利润释放的节奏如何"。两 skill 互补——剩余边际余额的规模直接影响内含价值中有效业务价值的厚度。
- 与 `operating-profit-decomposition`（营运利润拆解）的区别：营运利润拆解将四因子（摊销+投资收益+息差+偏差）全部展开，是全景式的利润来源分析；剩余边际分析聚焦于四因子中最核心、占比最大（~70%）的一项，深度分析其蓄水池动态和摊销机制。前者是"广度"，后者是"深度"。
- 与 `insurance-three-differences`（保险三差）的区别：保险三差是产品定价体系的利润来源（死差/费差/利差），属于"这张保单为什么赚钱"的微观解释；剩余边际是会计准备金体系的利润暂存库，属于"会计上为什么看不到这张保单赚的钱"的制度解释。三差是原因，剩余边际是结果。

## E — 可执行步骤 (Execution)

1. **定位数据源，提取剩余边际核心指标**
   - 从年报"寿险及健康险业务"分部的内含价值/营运利润披露章节，提取以下数据：
     - 期末剩余边际余额
     - 当期剩余边际摊销金额
     - 新业务贡献的剩余边际（或新业务价值，作为近似替代）
     - 预期利息增长
   - 计算关键比率：
     - 摊销率 = 当期摊销 ÷ 期初余额（反映利润释放速度）
     - 进水/出水比 = 新业务贡献 ÷ 当期摊销（反映蓄水池是在扩张还是收缩）
     - 剩余边际余额 ÷ 净资产（反映隐藏利润相对于账面净资产的倍数）
   - 完成标准：能说出"该险企剩余边际余额X亿，当期摊销Y亿，摊销率Z%，进水/出水比W倍"
   - 注意：仅中国平安和中国太保披露了完整的剩余边际明细（寿险分部），中国人寿和新华保险的披露颗粒度较粗，可能需要用新业务价值作为进水口的近似代入

2. **分析蓄水池动态——判断"水位"是在上升还是下降**
   - 将剩余边际变动拆为进水（新业务贡献+预期利息）和出水（摊销+其他调整）
   - 纵向看趋势：过去3-5年的进水/出水比是持续>1还是已经<1？趋势在收敛还是扩张？
   - 横向对比：同行中谁的剩余边际余额最大？谁的进水/出水比最高？
   - 判停条件：若进水/出水比连续两年<1且新业务价值持续负增长 → 剩余边际余额将见顶回落 → 未来2-3年营运利润增速将显著放缓 → 这是一个重大的基本面恶化信号，应立即提醒用户
   - 完成标准：能回答"这个蓄水池的水位还在上升吗？速度在加快还是放缓？"

3. **串联至盈利预测——估算剩余边际对利润的支撑力**
   - 计算剩余边际摊销 ÷ 税前营运利润（判断营运利润对摊销的依赖度）
   - 基于当前新业务价值增速，预判未来2-3年摊销增速的大致区间（新业务价值增速→滞后传导至摊销增速）
   - 敏感性分析：若剩余边际余额下降10%/20%，对营运利润的冲击有多大？
   - 完成标准：能给出一个类似"若未来3年NBV保持5%增长，摊销增速预计维持6-8%，营运利润增速约为7-9%"的定量预判区间
   - 警告：摊销增速的预测存在1-2年的滞后效应和摊销模式差异（保障型保单早期摊销快、储蓄型均匀），精确预测不可行，给出方向和区间即可，遵循"模糊的正确"原则

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 仅询问保险公司的 PEV（股价/内含价值）或 PB 估值倍数，不涉及准备金结构或利润来源的讨论——这是估值问题，非剩余边际分析的触发条件
- 分析财产险公司（如中国财险/人保财险）的盈利模式——财险保单通常为一年期，不存在长期保单的剩余边际概念，应采用综合成本率（COR）分析框架
- 用户只问保险公司的保费收入或市场规模排名，不关心利润的存储和释放机制——这是销售分析，非准备金分析
- 数据不充分（年报未披露剩余边际明细，且用户只持有中国人寿/新华保险等披露粗糙的公司数据）时强行给出精确结论——应明确告知数据局限性，仅基于新业务价值做定性推断
- 用户问的是"保险三差（死差/费差/利差）"这个产品定价层面的概念——剩余边际是会计准备金层面的概念，两者的分析框架和适用场景不同。应引导至保险三差分析 skill。

### 作者在书中警告的失败模式

- **流动性溢价操纵（ce08）**：会计利率（折现率）以750日国债曲线为基础外加"流动性溢价"，后者可由管理层自行判断。有时750日国债曲线上行但公司以"对未来利率悲观"为由施加负的流动性溢价，逆转折现率变动，导致需要增加准备金计提、减少当期利润（即减少剩余边际的释放）。这既可能是审慎之举，也可能是隐藏利润。投资者无法从外部判断动机，需要综合投资回报差异和营运偏差的趋势进行交叉验证。若公司连续多年实际投资收益显著高于5%假设且营运偏差持续为正，而会计利润却未同步增长，则存在通过准备金政策隐藏利润的嫌疑。

- **"即使不产生新业务，内含价值也增长"的价值陷阱（ce09）**：公司或卖方宣称"即使停售新单，内含价值仍按x%增长"，诱使投资者高估存量价值、低估新业务重要性。这种说法的致命缺陷：新业务增长放缓→剩余边际摊销增速滞后下降（1-3年滞后）→营运利润增速放缓→分红能力减弱→估值下移。失去了新业务的保险公司是"无源之水"，存量蓄水池终有耗尽之日。识别信号：新业务价值增速连续下滑+代理人持续萎缩+剩余边际进水/出水比趋于1。

- **营运假设保守度与利润释放的跷跷板效应（ce15）**：管理层可以通过"故意使用非常保守的营运假设→压低有效业务价值→持续释放正的营运偏差→误导投资者高估经营质量"。剩余边际的摊销模式在发单时锁定，摊销本身不受此影响，但剩余边际的"预期利息增长"和"营运偏差"都是基于精算假设的——假设越保守，预期利息越低（压低剩余边际增长），但后续营运偏差释放越容易为正。一端压下去，另一端翘起来，总利润量不变，但释放节奏被调整。需结合有效业务价值占内含价值比例的趋势来交叉识别。

### 作者的盲点 / 时代局限

- **数据截止2019-2020年，未覆盖低利率环境的冲击**：书中平安2019年剩余边际预期利息增长基于5%的利率假设。2020年后十年期国债持续下行（跌至2.5%左右），5%的假设能否维持存疑。若未来投资假设下调→预期利息增长贡献减少→剩余边际内生增速放缓→对进水口（新业务贡献）的依赖度被动提高。当前环境下使用此skill时，需主动降低对"预期利息增长"贡献的估计。
- **作者以中国平安为标杆，其他公司数据粗糙度被低估**：平安和太保在2017年率先采纳营运利润披露体系，剩余边际数据的颗粒度最细；但国寿和新华直到近期仍未按同等口径系统披露剩余边际明细。本书的剩余边际蓄水池分析范例完全以平安数据构建，投资者在分析其他公司时面临严重的数据不对称——这本身就是一个需要警惕的分析障碍。
- **IFRS17新保险合同准则（2023年生效）对剩余边际概念的重塑未覆盖**：IFRS17下"剩余边际"被重新命名为"合同服务边际（CSM）"，且计量和摊销规则有重大调整（如CSM吸收未来服务相关的假设变动、必须在每个报告期重新计量等）。书中基于旧准则（IFRS4/中国CAS）的分析框架在IFRS17生效后需要相应调整，不能全盘套用。

### 容易混淆的邻近方法论

- **混淆剩余边际与风险边际**：剩余边际是利润暂存，风险边际是保守计提的安全垫。前者是已经赚到的钱，后者是"万一出事"的预备金。两者都藏在准备金里，但风险边际随保单到期而消失（不释放为利润），剩余边际则逐期摊销释放。
- **将剩余边际摊销等同于"当期新赚的利润"**：剩余边际摊销是过去已赚、现在释放的钱，不是当期的新增盈利。保险公司的当期真正新增盈利主要体现在新业务价值（当年新签保单的预期利润折现）上。剩余边际摊销 + 新业务价值 → 才能完整描述保险公司的赚钱全景。
- **将剩余边际余额直接等同于"可分红利润"**：剩余边际虽然本质上已经是利润，但它在资产负债表上属于准备金（负债科目），受到偿付能力监管约束和保单持有人利益优先原则的限制，不能像净资产一样随心所欲地分红。分红能力取决于自由盈余（调整净资产中超过最低资本的部分），而非剩余边际余额。
