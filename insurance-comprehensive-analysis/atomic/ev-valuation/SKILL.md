# 内含价值估值框架 — DCF落在保险公司的完整方法论

## R — 原文 (Reading)

> 内含价值评估标准本质上是一种自由现金流折现法。在现金流的折现过程中，新业务价值扮演的是二传手的角色。某张保单通过各种假设测算出当年净现金流（实际上是净利润，但可以近似认为是现金流），这些现金流扣除一定的成本后，得出可以分配的自由净现金流，然后通过贴现的把戏，从未来穿越到现在，得到新业务价值。新业务价值在第二年变为有效业务价值，进入内含价值科目中，成为公司价值重要的部分。

> — 东先生，第5章 "新业务价值"

> 集团内含价值 = 集团的调整净资产 + 有效业务价值 + 归属于有效业务价值的持有偿付能力额度的成本

> — 东先生，第5章 "内含价值的构成"

> 调整净资产中负债的调整，主要是将会计准备金调整为内含价值准备金。为了让内含价值充分反映公司的价值，内含价值准备金使用的折现率将"采用最优估计"而非刻意保守的评估利率。折现率高了，调整后的负债也就低了，调整净资产自然也高出不少。

> — 东先生，第5章 "调整净资产"

> 1万元先被用 5% 的投资收益率假设推到保单生命的尽头，其产生的投资收益又被以 11% 的贴现率从尽头贴现回现在，来了一趟时间的旅程……1万元与3479元的差值6521元，即为持有偿付能力额度的成本。

> — 东先生，第5章 "持有偿付能力额度的成本"

> 内含价值增长 = 期初调整净资产每年贡献（增长率：3%~4%）+ 期初有效业务价值每年贡献（增长率：11%）（不考虑扣偿新业务价值）

> — 东先生，第5章 "内含价值的增长秘诀"

> 在估值上，老东认为……公司的价值，最合适的答案永远是：公司未来所有可以赚的自由现金流的折现值。因而，为了让我们的投资决策确定性更高，老东建议在行业和公司的思考上投入的时间的比重应该要大于估值，而不是颠倒过来。

> — 东先生，第5章 "第五章 内含价值评估标准"

> 一般会计准则的出发点是"从时间的起点去还原现在"……再看内含价值评估标准，颇有"从时间的终点来还原现在"的味道。

> — 东先生，第5章 "内含价值评估标准与一般会计准则"

## I — 方法论骨架 (Interpretation)

### 内含价值的公式解剖

内含价值(EV)是DCF落到人身险公司的具体形态，由三大构件组成：

```
内含价值 = 调整净资产(ANAV) + 有效业务价值(VIF) + 持偿成本(CoC，负数)
```

| 构件 | 含义 | 增长速率 | 角色 |
|------|------|----------|------|
| **调整净资产 (ANAV)** | 会计净资产经过资产→市值、负债→EV准备金的双调整 | 约3%~4%/年 | 清算保底价值 |
| **有效业务价值 (VIF)** | 存量保单(一年以前签发)未来利润按11%贴现回现在的总和 | 约11%/年（弹簧回弹） | 核心价值主体 |
| **持偿成本 (CoC)** | 持有最低资本的代价：资本按5%推到终点→按11%贴现回来，净折损 | 扣除项，随"最低资本-持偿成本"按11%增长 | 资本占用的负面项 |

### 新业务价值：价值增长的发动机

```
扣偿后新业务价值 = 新业务价值(NBV) + 归属于新业务的持偿成本(负数)
```

NBV是当年新签保单全生命周期净利润的贴现总和。它驱动内含着价值增长：
- **代理人端**：人均NBV越高，产能越强
- **产品端**：新业务价值率(NBV/首年保费)越高，产品盈利能力越强
- **NBV→第二年→VIF**：NBV第二年自动转为有效业务价值，从"增量"变为"存量"

### 内含价值的增长公式（实务近似）

```
年内含价值增长 ≈ 期初调整净资产 × 3%~4%
                 + 期初有效业务价值 × 11%
                 + 扣偿后新业务价值
                 + 投资回报差异 + 营运经验差异 + 假设变动 ± 其他
```

- **11%**来自风险贴现率的逐年回拨（弹簧回弹效应）
- **3%~4%**而非5%：因为免税资产、不可投资资产、税收等因素拉低了实际回报
- **投资回报差异**：实际投资收益率 vs 5%假设的差额，是"锅里的肉"，迟早会装进来
- **假设变动**：最大的"捣蛋鬼"——管理层调整贴现率或投资收益率可一次性影响内含价值数百亿

### 保险公司生命周期的内含价值增速规律

| 阶段 | 特征 | 内含价值增速 | 关键指标 |
|------|------|-------------|----------|
| **幼年期** | 自由盈余占比高，VIF占比低 | 靠近6% | 自由盈余占比 |
| **壮年期** | VIF占比高，NBV高速增长 | 靠近11%+NBV贡献 | 有效业务价值占比、NBV增速 |
| **老年期** | 利润大量留存，自由盈余臃肿 | 回落到6% | 分红率、资本利用率 |

> 逃脱"老年期宿命"的两个出路：高分红（把利润还给股东）或对有壁垒新业务的高投入（把利润重新变为VIF）。

### PEV：保险股估值的核心锚

```
PEV = 股价 ÷ 每股内含价值
```

- PEV < 1：市场认为内含价值假设过于乐观（清算价值折价）
- PEV ≈ 1：市场公允估值
- PEV > 1：市场认可公司持续创造超额NBV的能力
- 作者态度：PEV是锚但不迷信，行业和公司质地比估值更重要

## A1 — 书中的应用 (Past Application)

### 案例 1：偿二代切换 — 中国平安调整净资产暴增41%（c12）

- **问题**：2016年偿一代→偿二代的切换如何影响内含价值？
- **方法论的使用**：调整净资产的两阶段调整逻辑：(1)资产端：账面价值→市场价值；(2)负债端：会计准备金→内含价值准备金（折现率采用最优估计，剩余边际不再计入）。偿一代下调整净资产3279亿→偿二代下3482亿。
- **结论**：偿二代让保险公司摆脱"保守而刻板"的法定准备金束缚。剩余边际被认可为实际资本，保障型公司实现"轻资产"模式。调整净资产增幅反映的是被旧准则禁锢的真实价值释放。
- **结果**：作者强调——理解偿二代的内含价值逻辑，是看懂保险股真实价值的前提。

### 案例 2：新华保险2015年 — 三套准则下净资产差异巨大（c13）

- **问题**：同一家公司，三种准则三种净资产，投资者该信哪个？
- **方法论的使用**：新华保险2015年数据在三套计量标准下呈现三种净资产：会计净资产578亿、偿二代调整净资产768亿、偿一代调整净资产500亿。偿二代比会计口径高出189亿(+33%)，比偿一代高出268亿(+54%)。
- **结论**：这189亿差额本质上是被偿一代"禁锢"的保险公司真实价值——剩余边际。投资保险股必须问："你看的是哪个世界的净资产？"
- **结果**：这个案例成为作者论证内含价值优越性的经典素材——一般会计准则"失真"，内含价值才"公平"。

### 案例 3：4家险企NBV六年增速与价值率全景对比（c14, c15）

- **问题**：平安人寿NBV增速245% vs 新华仅99%，差距的根源是什么？
- **方法论的使用**：从两个维度拆解NBV增长——人均新业务价值（代理人端）和新业务价值率（产品端）。平安2019年NBV margin达43%，新华仅30%；平安人均NBV增速84%，新华负增长。
- **结论**：代理人渠道占比高 → NBV margin高 → 同等新单下NBV更高。这是"飞轮效应"——产能好→待遇好→吸引更优代理人→产能更好。强者恒强的逻辑在NBV指标上表现得淋漓尽致。
- **结果**：作者用数据打脸那些只看"新单保费规模"的投资者——规模不等于利润，NBV才是衡量增长质量的核心。

### 案例 4：平安人寿2016-2019年四年内含价值增长全景回溯（c18相关）

- **问题**：内含价值增长的"捣蛋鬼"能有多捣蛋？
- **方法论的使用**：逐回溯2016-2019年平安人寿内含价值增长过程。(1)2016年因投资收益率假设从5.5%→5%，EV一次性亏损421亿；(2)2017年因实际投资好于假设，投资回报差异贡献+302亿；(3)2018年无大变动但新增了明细披露；(4)2019年因税改清缴+85亿。四年间投资回报差异合计+496亿，已追回2016年一次性计提的421亿损失。
- **结论**：内含价值的增长过程不是千篇一律的风景。假设变动最高可影响EV的40%。但真正该关心的不是假设调节的"纸面盈亏"，而是公司能否保持优秀的实际投资回报——投资回报是"锅里的肉"，一块也跑不了。
- **结果**：作者用四年真实数据验证了"调整净资产×3~4% + 有效业务价值×11% + NBV"的预测公式，平均误差仅1.7%。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **保险股估值贵不贵？**：用户持有或关注某只保险股，想知道当前PEV处于什么位阶、是否低估/高估
2. **内含价值增长能持续吗？**：用户看到某公司EV持续增长，需要判断增长来自真实NBV创造还是假设游戏的"纸面繁荣"
3. **对比两家保险公司的价值**：两家公司EV规模相近或PEV相近，需要穿透数字看谁的增长质量更高
4. **贴现率/假设变动的影响评估**：某公司调整了贴现率或投资收益率假设，需要量化对内含价值的影响
5. **评估偿二代下的保险股**：用户需要理解偿一代→偿二代的转换对内含价值的"一次性释放"效应
6. **拆解NBV增长驱动**：某公司NBV增速快，需要拆解是来自人均产能提升、margin提升还是纯规模扩张

### 语言信号（用户的话里出现这些就应激活）

- "XX保险股现在估值贵不贵？"
- "PEV 0.5倍是不是被低估了？"
- "这家保险公司的内含价值增长能持续吗？"
- "内含价值这个数字可信吗？会不会是假设游戏？"
- "NBV增速为什么这么高/低？"
- "贴现率调了0.5%对EV有多大影响？"
- "偿二代和偿一代下内含价值差多少？"
- "有效业务价值占比高好还是调整净资产占比高好？"

### 与相邻 skill 的区分

- 与 **营运利润体系**：本 skill 聚焦内含价值评估标准（三个世界中的"DCF世界"），营运利润体系是嫁接一般会计准则和内含价值的"中间世界"。两者评估的"价值"口径不同，不可混用。
- 与 **保险三差/定价体系**：三差是产品定价体系的产物，与内含价值是"映射"关系而非"相等"关系。本 skill 在内含价值世界内做分析。
- 与 **偿付能力分析**：最低资本和持偿成本是将偿付能力体系桥接到内含价值的关键。理解偿付能力是先决条件，但本 skill 聚焦内含价值本身的估值与增长。

## E — 可执行步骤 (Execution)

### 步骤 1：建立EV构件表 — 从年报拆解内含价值

从目标保险公司年报"内含价值分析"章节提取以下数据（至少3年）：

| 科目 | 来源位置 | 说明 |
|------|----------|------|
| 调整净资产（人身险） | EV构成表 | 寿险及健康险业务调整后资产净值 |
| 有效业务价值 | EV构成表 | 1999年前+后的合计 |
| 持偿成本 | EV构成表 | 持有偿付能力额度的成本（通常为括号/负数） |
| 内含价值（人身险） | EV构成表 | 已扣偿的人身险内含价值 |
| 每股内含价值 | EV构成表或摘要 | 用于计算PEV |
| 一年新业务价值 | EV构成表 | 未扣偿的NBV |
| 扣偿后新业务价值 | EV构成表 | 已扣偿的NBV |
| 贴现率假设 | 主要假设 | 通常11%左右 |
| 投资收益率假设（永续） | 主要假设 | 通常5% |
| 调整净资产预计回报 | 内含价值变动表 | 中国平安2018年后单独披露 |
| 有效业务价值预计回报 | 内含价值变动表 | 同上 |
| 投资回报差异 | 内含价值变动表 | 实际vs假设的差额 |
| 营运经验差异 | 内含价值变动表 | 实际经验vs假设的差额 |
| 假设及模型变动 | 内含价值变动表 | 一次性调整项 |

**完成标准**：整理出一张3年×14项数据的时序表。

### 步骤 2：计算PEV — 判断估值位阶

```
PEV = 当前股价 ÷ 每股内含价值
```

**参考位阶**（结合市场环境和公司质地调整）：
- PEV < 0.5：深度低估（需确认不是实质价值毁灭）
- PEV 0.5~0.8：低估区间
- PEV 0.8~1.2：合理区间
- PEV > 1.5：溢价区间

**完成标准**：用"当前PEV vs 历史PEV分位 vs 同业PEV"三锚定位当前估值位阶。

**判停条件**：若PEV < 0.5且NBV连续负增长 → 这不是"低估"，这是"价值陷阱"。停止估值判断，先诊断NBV负增长的根源（代理人萎缩/渠道崩溃/产品竞争力丧失）。

### 步骤 3：检验EV增长的"纯度" — 区分真增长与假设游戏

用公式校验理论增长 vs 实际增长：

```
理论EV增长 = 期初调整净资产 × 3.5% + 期初有效业务价值 × 11% + 扣偿后NBV
```

与实际"内含价值变动表"中的增长对比，识别差异来源：

| 差异科目 | 正差异意味着 | 负差异意味着 |
|----------|-------------|-------------|
| 投资回报差异 | 实际投资好于5%假设 | 实际投资差于假设 |
| 营运经验差异 | 实际费用/赔付好于假设 | 实际坏于假设 |
| 假设及模型变动 | 假设调乐观（如降贴现率） | 假设调悲观（如降投资收益率） |

**重点关注**：假设及模型变动。若大幅为负（如平安2016年的-421亿），说明管理层在为过去的过度乐观"买单"。若大幅为正，需检查是否通过调参数"管理"内含价值。

**完成标准**：明确回答："本年内含价值增长中，多少来自真正的业务创造，多少来自假设调整？"

### 步骤 4：拆解NBV增长引擎 — 代理人端 vs 产品端

从年报中提取：
- 人均NBV = 扣偿后NBV ÷ 代理人数量
- 新业务价值率 = NBV ÷ 首年保费（规模保费口径）

**三年趋势判断矩阵**：

| 人均NBV | NBV margin | 诊断 | 投资含义 |
|---------|------------|------|----------|
| ↑ | ↑ | 量质双升，最理想 | ★ 强买入信号 |
| ↑ | ↓ | 靠堆人头冲量，未来不可持续 | ⚠️ 关注代理人增速放缓后的NBV断崖 |
| ↓ | ↑ | 在做"减法换质量"的转型 | ★ 谨慎乐观，看转型后人均是否止跌 |
| ↓ | ↓ | 全面恶化 | 🔴 回避 |

**完成标准**：明确回答"NBV增长的驱动力是来自产能提升还是规模扩张？这种驱动力可持续吗？"

### 步骤 5：压力测试 — 贴现率与投资收益率敏感性

从年报敏感性分析表中提取（以平安为例）：

| 参数变动 | NBV变动幅度 |
|----------|------------|
| 贴现率调低0.5%（11%→10.5%） | +4.7% |
| 贴现率调高0.5%（11%→11.5%） | -4.4% |
| 投资收益率每年+50bp | +10.3% |
| 投资收益率每年-50bp | -10.3% |

**关键比较**：
- 各公司贴现率是否一致？（平安11%、国寿10%、太保11%、新华11.5%）
- 若一国寿将贴现率从10%调至11%，NBV会缩水多少？（保守性差异的量化含义）
- 永续投资收益率5%在十年期国债利率2.5%的背景下是否仍可维持？

**完成标准**：输出"若贴现率上调0.5%+投资收益率下调50bp，EV/NBV将缩水约X%"的双重压力测试结果。

### 步骤 6：综合判断输出

按以下模板输出：

1. **估值位态**：当前PEV处于历史X%分位、同业中排第Y，估值结论（低估/合理/高估）
2. **增长质量**：本年EV增长Z%，其中理论增长贡献a%（纯业务创造），假设变动贡献b%（一次性），投资差异贡献c%（锅里的肉）
3. **增长引擎健康度**：NBV增速X%，人均NBV趋势Y，NBV margin趋势Z，结论（可持续/拐点/恶化）
4. **关键风险点**：不超过3条（如"贴现率假设低于同业→内含价值被高估""投资收益率假设在利率下行背景下面临下调风险""NBV增速高度依赖代理人增员而非产能提升"）
5. **跟踪关注点**：下一份年报应重点看什么指标验证判断

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **分析产险/财产险公司**：内含价值评估标准是为人身险（寿险/健康险/养老险）设计的。产险是短期业务，不存在有效业务价值的DCF计算需求
- **单一保险产品销售评估**：如"某款分红险划算吗"——这是产品分析而非公司估值，不适用内含价值框架
- **仅看一般会计准则财务报表时**：内含价值分析需要年报中单独披露的"内含价值分析"章节。仅凭利润表/资产负债表无法完成
- **保费增速的简单判断**："新单保费增长快=公司好"——这是规模视角，内含价值框架要求穿透到"NBV增长"这一利润视角
- **非上市保险公司的估值**：未上市公司不披露内含价值，只能用其他替代方法

### 作者在书中警告的失败模式

- **"即使不产生新业务，EV也有xx%增长"的价值陷阱** (ce09)：某些言论宣称存量保单的11%折现率回拨就能永续驱动增长。实际上NBV增长放缓→拖累剩余边际摊销→拖累分红率→传导到估值下降。失去NBV的保险公司如无源之水，终被沙漠吞噬。
- **2016年四家公司集体下调投资收益率假设** (ce13)：2016年前行业普遍使用5.5%假设，集体下调至5%导致EV一次性缩水。截至本书成书时十年期国债已跌至2.5%左右，但各公司仍维持5%假设。若再次下调，EV将再次受损。
- **营运假设保守度与EV的跷跷板效应** (ce15)：管理层可能通过"故意使用非常保守的营运假设→压低有效业务价值→持续释放正的营运经验差异→误导投资者高估经营质量"。一端压下去(VIF)另一端翘起来(营运偏差释放)，短期可误导只看单边指标的投资者。
- **过度追求精确计算的错误** (ce10)：内含价值的贴现率/投资收益率/死亡率等假设每个环节都有不确定性。五项指标准确率各80%，连乘后准确率崩至32%。作者建议"模糊的正确，胜过精确的错误"——用近似公式计算EV增长即可，把时间花在研究行业和公司上。

### 作者的盲点 / 时代局限

- **2020年以后利率环境进一步恶化**：本书成书于2021年，当时十年期国债约3.1%。截至2025年十年期国债约1.6~1.7%，远低于内含价值的5%投资收益率假设，假设的可靠性面临更严重的质疑。需要结合最新利率环境重新评估。
- **偿二代二期 (2022年实施)**：本书基于偿二代一期。偿二代二期对最低资本计算和实际资本认定都做了更严格的修订，可能影响持偿成本和内含价值
- **IFRS 17 (2023年实施)**：新会计准则改变了保险合同的计量方式，可能影响一般会计准则与内含价值的对应关系，本书未覆盖
- **平安/太保以外的公司披露不完善**：中国人寿和新华保险在内含价值变动表中的明细披露不如平安充分，部分分析步骤（如拆分有效业务价值预计回报）可能无法完整执行

### 容易混淆的邻近方法论

- **与PEV混淆为PB**：PEV是股价/内含价值（DCF概念），PB是股价/账面净资产（成本概念）。两者分母截然不同，不可直接对标。保险股PEV在1附近可能已是合理估值，而银行股PB在1以下可能仍高估。
- **与银行DCF/Dividend Discount模型混淆**：银行的DCF/DDM是基于实际可分配股利的折现，内含价值是基于精算假设的"理论利润"折现。两者都是DCF，但因假设丰富度差异巨大，内含价值天然假设更多，不确定性更大。
- **NBV margin vs 净利润率**：NBV margin=NBV/首年保费，分子是保单全生命周期利润的贴现值，分母仅是首年保费（时间维度不一致）。真正的"利润率"应该是新业务保费现值利润率=NBV/保费折现值（如友邦保险约10%），保险公司实际净利润率并不高。
