# 品牌分级与品牌力评估

## R (Reading) — 原文引用

> "品牌力就是消费者愿意为品牌付出溢价的程度。而这个溢价通常源于品牌的文化属性（最重要的是社交功能）和功能属性（最重要的是安全保障）。"
>
> "品牌1.0——品牌即认知；品牌2.0——品牌即优选；品牌3.0——品牌即品类；品牌4.0——品牌即身份。"
>
> "价格带就是品牌的生命线，降价容易提价难，品牌在保证一定销售规模的前提下，保持价格带稳定要远远比多卖些货重要。"

## I (Interpretation) — 方法论骨架

品牌力的本质是**定价能力**，分为三个递进层次：定价能力（能自主定价）→ 溢价能力（能卖得比同类贵）→ 涨价能力（能持续提价）。大多数有溢价的品牌只有前两层，涨价能力极其稀缺。

品牌分级体系提供了品牌进化路径图：
- **1.0 品牌即认知**：消费者知道品牌是做什么的，但谈不上偏好
- **2.0 品牌即优选**：在同类商品中会被优先选择，有了初步的"首选效应"
- **3.0 品牌即品类**：品牌与品类深度绑定，"提到品类就想到品牌"，市占率高，有较高溢价
- **4.0 品牌即身份**：拥有即身份象征，产品是同品类中最贵的，溢价极高
- **4.0+ 超身份**：产品形成稳定二级市场流通，二手成交价超过品牌定价

跃迁关键：2.0→3.0 靠功能属性（品类强绑定+质量信任），3.0→4.0 靠文化属性（社交功能+身份认同）。绝大多数消费品品牌无法跨越 3.0。

品牌力评估的三个核心指标：**规模**（消费者真金白银的投票）、**增速**（品牌"做对了事"的即时信号）、**定价能力**（品牌力厚度的标尺）。一万个可能成功的理由，不如规模已经做起来有说服力。

## A1 (Past Application) — 书中案例

1. **茅台 = 品牌 4.0+**：飞天茅台定价 1499 元，经销商调货价约 4000 元（2 倍+），有稳定二级市场和做市商，具备"通货"属性。净利率远超实物商品天花板。
2. **波司登 = 品牌 3.0→4.0 跃迁**：与羽绒服品类深度绑定（3.0），2018 年品牌升级后主力价格带从 <1000 元升至 1000~3000+ 元，高端款售罄。靠功能性赛道+国际品牌打开价格天花板+自身制造/设计实力完成跃迁。
3. **薇诺娜 = 品牌 3.0**：皮肤学级护肤品龙头，通过"功能性"标签与品类强绑定，市占率远高于普通护肤品龙头。

## A2 (Future Trigger) — 触发情境

- 分析任意消费品公司时，需要评估其品牌护城河深度
- 判断某品牌是否具备持续提价能力
- 比较同行业多个品牌的竞争力差异
- 在投资决策中需要给品牌力赋值
- 跟踪品牌升级/降级的信号（价格带变化、二手市场价格变化）

## E (Execution) — 执行步骤

1. **定位品牌级别**：根据品牌的市场认知、终端价格与同类对比、消费者提及品类的第一联想，判断品牌处于 1.0~4.0+ 哪个级别
2. **评估跃迁潜力**：分析品牌所在品类的功能属性和文化属性强度。功能属性品类（羽绒服/护肤品/运动鞋）更容易达到 3.0；文化属性强的品类（白酒/皮具/手表）才有可能达到 4.0
3. **跟踪核心信号**：监控品牌的终端售价变化（是否稳中有升）、二手市场交易情况（出现溢价转卖是 4.0+ 信号）、新品价格带是否持续上移

## B (Boundary) — 边界与不适用

1. **不可用于非品牌类消费品公司**：渠道公司（京东/永辉）和供应链公司（申洲国际）不适用此分级体系，它们有不同的壁垒来源
2. **品牌力 ≠ 投资价值**：1.0~2.0 级品牌也可以是优秀投资标的（如农夫山泉 = 省心型品牌 = 2.0 级，但长期回报优秀），分级仅反映定价能力路径，不代表投资优劣
3. **品牌跃迁不是必然发生的**：3.0→4.0 是极少数品牌的成果，大多数品牌会在某个级别停留甚至退化。不可因为品牌"有潜力"就高估其跃迁概率
4. **国产品牌替代逻辑有风险**：书中乐观估计国产品牌崛起，但实际上同一赛道的国产品牌可能陷入恶性价格战，品牌力在此过程中会被侵蚀
5. **不适用于科技产品短期判断**：科技消费品（如手机）品牌力受技术迭代周期影响更大，品牌分级需结合技术周期调整
